# Systemic Lupus Erythematosus (`systemic_lupus_erythematosus`)

- 動力學類型：`chronic_relapsing`
- 患者數：**200**，平均年齡 39.6 歲，老年 6 位
- 90 天 flare 平均：**2.73**
- 模型 MAE 平均：**0.176**，flare 召回平均 64%，準確率平均 77%
