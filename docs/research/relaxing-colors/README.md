# 放鬆配色研究筆記（Relaxing Colors Research）

> **用途**：個人研究參考資料，**不用於 production 部署**。
> **建立日期**：2026-04-13
> **研究問題**：大眾普遍認為「放鬆」的配色有哪些？是否適合用於醫療/醫病溝通類 App？

---

## 一、最被廣泛認可的放鬆色（Top 3）

### 1. 藍色 Blue（首選療癒色）
- **生理效應**：降低心率、血壓和呼吸頻率，減輕焦慮
- **聯想**：天空、海洋 → 開闊與寧靜
- **實證案例**：日本部分火車站改用藍色 LED 燈後，自殺率下降 84%
- **建議色票**：
  - 天空藍 `#A8D8EA`
  - 莫蘭迪藍 `#B5CDD8`
  - 鴨綠藍（teal）`#7FB3B5`

### 2. 綠色 Green（療癒/平衡色）
- **生理效應**：降低緊張、調節自律神經系統
- **聯想**：植物、自然 → 安全感、再生
- **實證**：診所候診區、電視節目休息室常用，能有效減輕壓力
- **建議色票**：
  - 鼠尾草綠 `#B8C9A8`
  - 薄荷綠 `#C8E6C9`
  - 抹茶綠 `#A4B494`

### 3. 大地色 Earth Tones（中性放鬆）
- **生理效應**：穩定邊緣系統（情緒中樞）、減少視覺刺激
- **聯想**：木、土、石 → 歸屬感、溫暖
- **建議色票**：
  - 米杏 `#EAE0D5`
  - 燕麥 `#D6CCC2`
  - 暖灰 `#C9C0B6`

---

## 三、設計原則（醫療 App 適用）

1. **冷色系做背景**，營造整體寧靜感（藍/綠/灰）
2. **避免高飽和、高對比**，會引起焦慮
3. **限制色彩數量**：建議主色 1 + 輔色 2 + 中性色 2
4. **柔和、低飽和的「莫蘭迪色系」** 最被認為放鬆
5. **跨文化測試**：紅色在亞洲為喜慶、在西方為警告，需注意
