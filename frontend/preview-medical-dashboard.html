系統元素 v4 新加 / 用法
KPI 數據條頂部 4 卡：血壓 / 血糖 / 用藥依從 / 電量，每張有 sparkline + delta + last-time + 狀態 pill
Sub-tab頁面內：今天 / 本週 / 本月 / 全部 — 每個 tab 有計數 badge
表格化清單Sortable header（時間 ↑）+ filter chip row + metadata footer（規則引擎版本 / 更新時間）
Pill / Badge system統一 7 種：ok / warn / danger / info / teal / mute / rose — 顯示準時 / 略高 / 未到 / 已記 等狀態
右側 Activity feed即時時間軸 · 9 筆活動 · 每筆含 source 標籤（MANUAL / BT_DEVICE / AI / SYS / REMINDER）
Severity grid（5 級分級醫療）水平 5 格：自我照護 / 診所 / 區域 / 醫學中心 / 急診，每格有色點 pip + selected highlight
Source / Metadata footer卡片底部 source-row：規則版本、conf 分數、verified 標、更新時間 — 「為什麼」可追溯
Monospace 用法只用於：數字 / 時間戳 / ID / 版本 / 計數 — 跟內文 sans-serif 分開
⌘K 搜尋Sidebar 頂部全站搜尋 + 鍵盤捷徑提示
Topbar status pill「已同步 · 2 秒前」帶綠脈衝點，告訴使用者系統健康
