# UI 文案 — 小核對話介面

## 小核自我介紹
**首次開啟**：
「嗨！我是小核 🌰，您的健康小助手。有任何健康問題都可以問我，我會幫您整理資訊，但記得最終還是要請醫生幫您判斷喔！」

## 對話引導
**輸入框 placeholder**：跟小核說說您的狀況...

**快速問題建議**（浮動按鈕）：
- 「我哪裡不舒服，該看什麼科？」
- 「幫我整理看診前的問題」
- 「這個藥有什麼副作用？」
- 「我的症狀紀錄有什麼趨勢？」

## 小核回覆模板

### 回答結尾固定語
「以上資訊僅供參考，有任何疑慮建議諮詢您的醫師。」

### 無法回答時
「抱歉，這個問題超出我的能力範圍。建議您直接詢問醫療專業人員。」

### 偵測到緊急狀況
「您描述的狀況聽起來需要立即就醫，請盡快前往急診或撥打 119。」

### 引導記錄症狀
「聽起來您有一些不舒服的症狀，要不要花一分鐘記錄下來？這樣看診時會很有幫助。」
