# 多重用藥：藥袋大整理與重複用藥

## 一句話懂

老人**多重用藥（≥ 5 種）**：跌倒、副作用、認知下降、住院、死亡風險都升。**定期與藥師整合審視**可救命。

## 想多了解

### 多重用藥的問題

**多 = 副作用多**
- 藥物交互作用
- 累積副作用
- 跌倒風險

**抗膽鹼負擔**
- 多種藥共有抗膽鹼效應
- 累積 → 譫妄、失智、便秘、跌倒
- 評估 ACB Score

**遵藥性差**
- 多藥多時間 → 易漏服
- 影響療效

**經濟負擔**

**互相矛盾**
- 多醫師多開藥
- 重複用藥

### 「處方瀑布」

**典型循環**
- 藥 A 副作用 → 加藥 B 治副作用 → 藥 B 副作用 → 加藥 C ...

**例**
- 降壓藥 → 頭暈 → 暈眩藥
- 利尿劑 → 高尿酸 → 痛風藥
- 抗精神病 → 動作障礙 → 抗巴金森

### Beers Criteria（老年人不適藥物）

**避免使用**
- 苯二氮平類（跌倒、認知）
- 某些抗精神病藥
- 抗組織胺第一代（Diphenhydramine）
- 三環抗憂鬱
- 部分非選擇性 NSAID

**老人替代**
- Z 類（也要小心）→ 改 CBT-I
- 第二代抗組織胺
- SSRI 取代三環

### 藥袋整合審視

**何時做**
- 用藥 ≥ 5 種
- 多家醫院看診
- 出院後
- 出現副作用
- 跌倒、認知變化

**哪裡做**
- 醫院藥師諮詢
- 居家藥事服務
- 藥局藥師

**評估什麼**
- 哪些可降階 / 停藥
- 是否重複
- 交互作用
- 適合老人嗎
- 服藥能力

### 「降階」（Deprescribing）

**原則**
- 不必「越多越好」
- 利益 vs 害評估
- 漸減觀察
- 與醫師討論

**常可降階**
- 預期壽命短的預防藥（Statin、Bisphosphonates）
- 過量降壓藥（造成低血壓）
- 過量降糖藥（造成低血糖）
- 長期 PPI（沒指徵）
- 重複用藥

### 雲端藥歷

**健保資源**
- 健保卡查
- 健保署 APP
- **多家醫院都會看**

**用法**
- 每位醫師看
- 避免重複
- 看交互作用

### 隨身藥單

**準備一份**
- 所有藥名（中英）
- 劑量、頻率
- 過敏史
- 看不同醫院帶

### 居家藥事服務

**長照 2.0 + 藥師居家服務**
- 藥師到府整理
- 評估遵藥性
- 教導使用

### 老人遵藥性技巧

**藥盒**
- 一週分裝
- 早晚分開
- 大字版

**鬧鐘提醒**
- 手機 / 智慧藥盒

**家人協助**
- 視力差、認知差者更需要

### 何時找醫師「降階」

**討論時機**
- 預期壽命短
- 副作用大
- 經濟負擔
- 認知開始下降
- 跌倒反覆

## 什麼時候要找醫師 / 藥師
- 用藥 ≥ 5 種
- 反覆副作用
- 跌倒
- 認知變化
- 出院 / 換醫師

> 多重用藥**整合審視可降害**——「降階」不是不負責，是聰明照顧。
