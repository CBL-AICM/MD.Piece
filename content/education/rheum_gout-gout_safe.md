# 痛風可以放心吃的：乳製品、咖啡、櫻桃、足量水

## 一句話懂

痛風飲食常被講成「什麼都不能吃」——其實**有一份清單可以放心、甚至幫忙**。把這幾項加進日常，比什麼都不吃更實際。

## 想多了解

### 1. 乳製品（特別低脂）

**為什麼好**
- 乳清蛋白促進尿酸排出
- 抗發炎
- **每天兩份**可降痛風風險 50%

**選什麼**
- 低脂牛奶
- 無糖優格
- 起司

**注意**：高脂乳品效果略差

### 2. 咖啡

**為什麼好**
- 黃嘌呤氧化酶抑制
- 規律喝者尿酸較低
- 一天 2–3 杯內合理

**注意**
- 黑咖啡最佳
- 不要加糖
- 心律不整、失眠者酌量

### 3. 櫻桃

**證據**
- 多項研究顯示：**減少 35% 發作風險**
- 每天 10–12 顆即可
- 鮮櫻桃 / 濃縮汁皆可
- 富含花青素 → 抗發炎

**其他抗發炎莓果**
- 草莓、藍莓、蔓越莓
- 黑莓

### 4. 水（最便宜的藥）

**目標**：**每天 ≥ 2 公升**

**好處**
- 稀釋尿酸
- 幫腎排尿酸
- 預防腎結石

**心衰、嚴重腎病者**：依醫師建議

### 5. 全穀類

**燕麥、糙米、全麥**
- 纖維高、低升糖
- 可放心
- 比白飯好

### 6. 蔬菜（含「高普林蔬菜」）

**可放心吃**
- 菠菜、蘆筍、白花椰
- 香菇、豆腐
- **植物性普林不會誘發痛風**（研究證實）

### 7. 雞蛋

- 蛋白：低普林
- 蛋黃：適量
- 一天 1–2 顆 OK

### 8. 健康脂肪

**Omega-3**
- 鯖魚、秋刀魚（少量）
- 亞麻籽
- 核桃
- 抗發炎

**橄欖油**
- 地中海飲食一部分

### 9. 維生素 C

**證據**
- 500 mg/day 可降尿酸
- 柑橘類、奇異果、芭樂
- 也可補充劑

### 10. 全植物蛋白

**豆腐、豆漿、豆類**
- **植物性蛋白安全**
- 取代部分肉類
- 富含纖維

## 一週飲食範例

| 時段 | 安全選擇 |
| :-- | :-- |
| 早 | 低脂牛奶 + 燕麥 + 莓果 |
| 點心 | 黑咖啡 + 雞蛋 |
| 午 | 全麥三明治 + 雞肉 + 沙拉 |
| 點心 | 櫻桃 / 奇異果 |
| 晚 | 糙米飯 + 豆腐 + 大量蔬菜 + 少量魚 |

## 可以怎麼做

- 🥛 每天兩份乳製品
- ☕ 早晨黑咖啡
- 🍒 季節加櫻桃
- 💧 水壺隨身
- 🍞 主食改全穀
- 🥗 餐餐蔬菜
- 📓 飲食日記比較症狀

## 什麼時候要找醫師

- 飲食調整後仍反覆發作
- 想做飲食諮詢
- 合併其他疾病需個別化飲食

> 痛風飲食**不是「不能吃」，是「換成什麼吃」** ——**好好吃** 也能控制好痛風。
