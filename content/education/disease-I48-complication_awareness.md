# 心房顫動併發症：中風、心衰、認知下降

## 一句話懂

AFib 不治療**最大威脅是中風**（風險升 5 倍）+ **心衰** + **認知下降**。**抗凝血藥 + 控心率**可大幅降低。

## 想多了解

### 缺血性中風
- AFib 患者 5 倍風險
- 多大面積栓塞型中風
- 預後差於其他類型中風
- 抗凝血藥可降 60% 風險
- **CHA₂DS₂-VASc 評分**指引用藥

### 心衰竭
- AFib 可加重 / 誘發心衰
- 心搏量降 20–30%
- 控好心率 / 心律可改善
- 雙向關係（心衰也誘發 AFib）

### 認知下降 / 失智
- AFib 患者失智風險升
- 微小腦栓塞累積
- 抗凝血藥可降風險
- 規律治療保腦

### 心血管事件
- 心肌梗塞風險升
- 多源於共同危險因子（高血壓、糖尿）
- 整合管理

### 出血（抗凝血藥相關）
- 顱內出血最致命
- 胃腸出血
- 鼻血、瘀青
- 個別化評估
- HAS-BLED 評分

### 心動過速性心肌病變
- 長期心率太快 → 心肌弱化
- 控好心率可恢復

### 預防併發症的核心
- **抗凝血藥規律服**（不可漏）
- 控心率 < 110
- 控血壓 < 130/80
- 治療共病
- 戒菸戒酒
- 監測出血

### 篩檢時程

| 檢查 | 頻率 |
| :-- | :-- |
| 心電圖 | 每年或症狀變化 |
| 抽血（肝腎、INR） | 每 3–6 個月 |
| 心臟超音波 | 每 1–2 年 |
| CHA₂DS₂-VASc | 每年重評 |

### 左心耳封堵術（LAA closure）
- 出血高風險者替代抗凝血藥
- WATCHMAN device
- 健保條件給付

## 什麼時候要找醫師
- 出現中風跡象
- 抗凝血藥相關出血
- 心衰症狀
- 認知變化
- CHA₂DS₂-VASc 上升

> AFib **併發症可預防**——規律抗凝血 + 共病管理 = 保腦保心。
