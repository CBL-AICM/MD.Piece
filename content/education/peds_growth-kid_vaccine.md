# 兒童疫苗時程：公費 + 自費完整時程表

## 一句話懂

台灣兒童疫苗政策超完整——**公費照時程打**就涵蓋大部分必要疫苗。**自費部分**（輪狀、A 肝加強、流感）建議盡量補上，可大幅減少就醫。

## 想多了解

### 公費疫苗時程（重點）

| 年齡 | 疫苗 |
| :-- | :-- |
| **出生 24 小時內** | B 肝免疫球蛋白（高風險母）、B 肝第 1 劑 |
| **出生 5 天內** | 卡介苗 |
| **1 個月** | B 肝第 2 劑 |
| **2、4、6 個月** | 5 合 1（白喉、百日咳、破傷風、Hib、小兒麻痺） + 13 價肺炎鏈球菌 |
| **6 個月** | B 肝第 3 劑、流感（每年） |
| **12 個月** | 麻疹腮腺炎德國麻疹（MMR）、水痘、A 肝第 1 劑、13 價肺炎鏈球菌補強 |
| **18 個月** | 5 合 1 補強 |
| **小一前** | MMR 第 2 劑、水痘第 2 劑、4 合 1 補強、日本腦炎 |
| **小六** | 子宮頸癌疫苗 HPV（女）|

### 自費疫苗（建議補）

- **輪狀病毒**（口服，6 個月前完成）— 可大幅減少嚴重腸胃炎
- **腸病毒 71 型**（2024 公費，2 個月起）
- **流感**（每年 1 次，6 個月以上）
- **A 肝加強**（公費已涵蓋）

### 疫苗安全
- 副作用多為輕微：發燒、注射部位紅腫
- 嚴重過敏（< 1/百萬）
- **疫苗不會造成自閉症**（這是被推翻的偽研究）
- 早產兒按實際年齡打，不需延後

### 注意事項
- 急性發燒 > 38°C 暫緩
- 免疫功能低下者諮詢醫師再打活性疫苗
- 對雞蛋過敏可打流感疫苗（除嚴重過敏）
- 健康手冊每次帶（記錄完整）

## 可以怎麼做

- 📋 把疫苗卡放在固定位置，每次帶
- 📅 預約下一次接種時就排好
- 📞 健保 APP 可查疫苗紀錄
- 🌡 接種後 48 小時內可能輕微發燒，正常

## 什麼時候要找醫師

- 接種後 48 小時內發燒 > 39°C
- 嚴重哭鬧 > 3 小時
- 接種部位嚴重紅腫超過手掌大
- 出現抽搐
- 任何過敏反應
