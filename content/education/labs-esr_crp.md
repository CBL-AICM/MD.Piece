# ESR / CRP：發炎指標的解讀與限制

## 一句話懂

ESR、CRP 都是**非特異性發炎指標**——告訴你「身體在發炎」，但**不告訴你哪裡發炎**。要配合臨床判斷。

## 想多了解

### CRP（C 反應蛋白）
- 肝臟在發炎時分泌
- **數小時內**升高
- 治療反應**快速下降**
- 細菌感染、組織損傷敏感
- 高敏 hs-CRP 用於心血管風險

**正常**：< 5 mg/L（一般 CRP）

**升高解讀**
- < 10：輕度（病毒、肥胖、慢性發炎）
- 10–100：中度（細菌感染、發炎、自體免疫）
- > 100：嚴重（菌血症、組織壞死）

### ESR（紅血球沉降速率）
- 紅血球沉降的速度
- 受**纖維蛋白原**影響
- **緩慢上升、緩慢下降**
- 反映「過去幾天到幾週」的發炎

**正常**
- 男 < 15、女 < 20 mm/hr
- 老人略高

**升高原因**
- 感染
- 自體免疫疾病
- 惡性腫瘤
- 妊娠
- 貧血
- 多發性骨髓瘤

**極高 (> 100)**
- 多發性骨髓瘤
- 巨細胞動脈炎
- 嚴重感染
- 惡性腫瘤

### CRP vs ESR

| | CRP | ESR |
| :-- | :-- | :-- |
| 反應速度 | 快（數小時） | 慢（數天） |
| 治療後 | 快下降 | 慢下降 |
| 影響因子 | 較少 | 多 |
| 急性 | 較準 | 較不靈敏 |
| 慢性 | 不夠 | 更佳 |

**多兩者一起看**

### 限制
- **不特異**：發炎、感染、腫瘤都會升
- 有些疾病不靈敏（如 SLE：CRP 升要懷疑感染）
- 影像 + 抗體更能定診

### 何時抽
- 不明發燒
- 不明發炎症狀
- 自體免疫疾病追蹤
- 手術前評估
- 治療反應評估

### hs-CRP 心血管應用
- 較敏感版本
- < 1：低風險
- 1–3：中
- > 3：高
- 配合 LDL 評估

## 什麼時候要找醫師
- ESR / CRP 持續異常
- 不明原因升高
- 已治療仍未下降

> 發炎指標**像火警警報**——告訴你有火但不告訴你在哪。**配合臨床看才有意義**。
