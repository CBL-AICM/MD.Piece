# COPD 症狀辨識：晨咳痰多、活動後喘、急性惡化

## 一句話懂

COPD **典型表現**：晨咳痰多 + 活動後喘 + 反覆呼吸道感染。**急性惡化警訊**（痰變黃綠、喘加重）要立刻處理。

## 想多了解

### 慢性穩定症狀
- **晨咳**（白色痰）
- **運動後喘**（漸進式）
- **胸悶、胸口緊**
- 容易疲倦
- 反覆「感冒」

### 急性惡化（Exacerbation）警訊
- **喘比平常加重**
- **痰量增加**
- **痰變黃綠** 或膿樣
- 發燒
- 胸悶加重
- 意識變化（低氧）

### 為什麼急性惡化要重視
- 每次惡化會永久損失部分肺功能
- 嚴重者需住院
- 增加死亡率
- **早期介入**可降惡化嚴重度

### MRC 喘息分級
- 0：無症狀
- 1：劇烈運動才喘
- 2：平地快走或上小坡喘
- 3：100 公尺平地必須停下喘氣
- 4：穿衣服都喘

### 嚴重度警訊
- 嘴唇、指甲發紫（缺氧）
- 不能說完整句子
- 心率 > 120 / 呼吸 > 30
- 胸口反向呼吸（用輔助肌）
- 意識變化

### 與氣喘區別

| | COPD | 氣喘 |
| :-- | :-- | :-- |
| 起病年齡 | 中老年 | 任何年齡 |
| 抽菸史 | 多有 | 不一定 |
| 變化 | 逐漸進展 | 時好時壞 |
| 過敏 | 少 | 常 |
| 對藥反應 | 部分 | 顯著 |

### 自我監測
- **PEF**（最大呼氣流量）
- 紀錄每日喘程度、痰量、活動耐受
- COPD 行動計畫（綠/黃/紅燈）

### 居家氧氣監測
- 血氧機（指尖型）
- SpO2 < 90% 警訊

## 什麼時候要找醫師
- 喘比平常持續加重
- 痰變黃綠
- 發燒 > 38°C
- 急救藥效果差
- SpO2 < 90%

> COPD **學會自我監測**——早期辨識急性惡化，可在家或門診處理，避免住院。
