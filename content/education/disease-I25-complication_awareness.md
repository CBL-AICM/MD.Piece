# 冠心病併發症：心肌梗塞、心衰竭、心律不整

## 一句話懂

冠心病不治療會走向**心肌梗塞 → 心衰 → 心律不整**——但**規律藥物 + 控三高 + 戒菸**，多數可避免或延緩。

## 想多了解

### 心肌梗塞
- 完全堵塞 → 心肌缺氧死亡
- 急性處理（通血管）
- 後續藥物、復健
- 規律治療下二次梗塞風險可大降

### 心衰竭
- 心肌受損 → 收縮力下降
- 喘、水腫、疲倦
- 射血分數 EF 下降
- ACEi/ARB/ARNI、β-blocker、SGLT-2 都重要

### 心律不整
- **心房顫動**：缺血性心臟病常合併
- **心室頻脈、心室顫動**：致命，需電擊
- 嚴重者植入心律調節器或 ICD

### 突發性心因猝死
- 多源於心室顫動
- 長期心臟病者風險高
- ICD 可救命

### 預防併發症的核心
- **規律服藥不要停**
- 控好血壓 < 130/80
- LDL < 70（次發預防）
- HbA1c < 7%（糖尿病）
- 戒菸
- 規律運動 + 心臟復健
- 體重、壓力管理

### 篩檢時程

| 檢查 | 頻率 |
| :-- | :-- |
| 心電圖 | 每年或每次回診 |
| 抽血（三高） | 每 3–6 個月 |
| 心臟超音波 | 每年或必要時 |
| 心肌灌流 | 症狀變化時 |
| 心導管 | 必要時 |

### 心臟復健（被低估的治療）
- 4–12 週課程
- 監督下運動 + 衛教
- 健保給付
- 證實降死亡率
- **應主動詢問**

### 心理影響
- 心肌梗塞後憂鬱率高
- 影響預後
- 必要時抗憂鬱藥（注意藥物交互）

## 什麼時候要找醫師
- 症狀加重或改變
- 出現新症狀（喘、水腫、心悸）
- 想評估心臟復健
- 想評估 ICD 必要性
- 心理問題

> 冠心病**早治療早預防**——併發症不是必然。
