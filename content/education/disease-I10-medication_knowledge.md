# 高血壓用藥：四大類藥物與選擇原則

## 一句話懂

高血壓藥**不是「吃了停不下來」**——而是**幫你保護心、腦、腎**。四大類各有特色，依**共病、年齡、種族**選擇。

## 想多了解

### 四大類

**1. ACEi / ARB（最常用）**
- 作用：擴張血管、保護腎
- ACEi：Captopril、Enalapril、Lisinopril
- ARB：Losartan、Telmisartan、Olmesartan
- 副作用：ACEi 乾咳（10%）、血鉀升高
- 適合：糖尿病、心衰、慢性腎病
- 孕期禁用

**2. 鈣離子通道阻斷劑（CCB）**
- 作用：擴張血管
- Amlodipine、Felodipine、Nifedipine
- 副作用：踝部水腫、頭痛、潮紅
- 適合：老人、單純高血壓、心絞痛
- 與葡萄柚有交互作用

**3. 利尿劑**
- Thiazide 類：HCTZ、Indapamide
- 副作用：低血鉀、升尿酸（痛風者慎用）
- 老人、單純高血壓有效

**4. β-blocker**
- Bisoprolol、Atenolol、Carvedilol
- 副作用：心率慢、疲倦
- 適合：合併心絞痛、心衰、心房顫動
- 不再是單純高血壓首選

### 進階藥
- α-blocker：攝護腺肥大同時用
- 中樞作用：頑固性高血壓
- Aldosterone 拮抗劑：心衰、頑固性

### 選藥原則

| 共病 | 首選 |
| :-- | :-- |
| 糖尿病 | ACEi/ARB |
| 慢性腎病 | ACEi/ARB |
| 心衰 | ACEi/ARB + β-blocker |
| 心房顫動 | β-blocker |
| 痛風 | 避利尿劑、用 ARB Losartan |
| 老人單純收縮 | CCB 或 Thiazide |
| 黑人 | CCB 或 Thiazide |

### 服用原則

**時間**
- 多在固定時間
- 部分研究：睡前服可較好控夜間血壓
- 利尿劑早上吃（半夜不會起來尿）

**飲食**
- 葡萄柚會與部分 CCB 交互作用（避免）

**漏吃**
- 想到立刻補
- 接近下次跳過

**不要自己停**
- **突然停會反彈**
- 收縮壓可能爆高、心血管事件

### 副作用通報
- 頭暈：可能血壓降太低
- 持續乾咳：考慮換 ARB
- 踝水腫：CCB 常見，可暫減量
- 性功能：β-blocker 有時影響

### 健保涵蓋
- 多數降壓藥健保給付
- 學名藥便宜

## 什麼時候要找醫師
- 血壓不達標
- 副作用明顯
- 想評估換藥
- 出現急症症狀
- 其他疾病新發（共病改變）

> 高血壓藥**保護器官 30 年**——按時規律是給未來的禮物。
