# 發炎性下背痛：晨僵、活動後改善、夜間痛醒

## 一句話懂

AS 的下背痛**不像閃到腰**——是**慢慢來、越休息越痛、活動後反而舒服**的「發炎性」下背痛。年輕男性**清晨痛醒**就要懷疑。

## 想多了解

### 發炎性 vs 機械性下背痛

| | 發炎性（AS） | 機械性 |
| :-- | :-- | :-- |
| 起病 | < 40 歲、漸進 | 任何年齡、突發 |
| 持續 | > 3 個月 | 多 < 6 週 |
| 晨僵 | > 30 分鐘 | < 15 分鐘 |
| 休息 | **加重** | 改善 |
| 活動 | **改善** | 加重 |
| 夜間 | **痛醒** | 不太痛醒 |
| 對 NSAID | 反應好 | 反應普通 |

### ASAS 發炎性下背痛標準（4/5 即陽性）
1. 40 歲前發病
2. 漸進性起病
3. **活動後改善**
4. 休息不會緩解
5. **夜間痛**（特別下半夜）

### 受影響部位
- **薦腸關節炎**（典型，臀部痛）
- **腰椎、胸椎、頸椎**漸進往上
- **肌腱附著點炎**：阿基里斯腱、足底筋膜
- **末稍關節**：膝、腳踝
- **關節外**：葡萄膜炎、銀屑病、發炎性腸道

### 警示症狀
- **下背痛 > 3 個月**
- **晨僵超過 30 分鐘**
- **臀部痛**（單側或交替）
- **夜間痛醒**特別下半夜
- 對 NSAID 反應好
- 家族 AS 史

### 高風險族群
- **年輕男性**（女男 1:3）
- 15–30 歲發病
- HLA-B27 陽性
- 家族史

### 為什麼要早診斷
- 拖延治療 → 脊椎強直、駝背
- 早治療能**保住活動度**
- 平均診斷延遲 5–7 年（很可惜）

## 可以怎麼做

- 📓 寫疼痛日記（時間、誘因、緩解因素）
- 🚶 規律運動（核心、伸展）
- 🚭 戒菸（強烈影響進展）
- 🩺 持續 3 個月就診
- 📋 帶舊抽血、影像

## 什麼時候要找醫師

- 下背痛 > 3 個月、< 40 歲
- 晨僵 > 30 分鐘
- 夜間痛醒
- 家族 AS、銀屑病、發炎性腸道病史
- 反覆紅眼睛（葡萄膜炎）

> AS **不是「老化」**——年輕人的下背痛要當回事。**早診斷 + 規律運動 + 適當用藥**可以正常生活。
