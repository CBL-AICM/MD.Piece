# SLE 抽血指標：ANA、anti-dsDNA、anti-Sm、補體 C3/C4

## 一句話懂

SLE 抽血**不是一次驗完**——是**長期追蹤的地圖**。各項指標代表不同訊息：診斷、活動度、預後。

## 想多了解

### 診斷必驗

**ANA 抗核抗體**
- > 99% SLE 陽性
- **陰性幾乎可排除 SLE**（除少數 SSA 陽性）
- 健康人 5–10% 也陽性 → 不能單看

**anti-dsDNA**
- SLE **特異性高**
- 與**狼瘡腎炎、活動度**相關
- 治療後應下降
- 高效價持續 → 預示腎炎

**anti-Sm**
- 雖少見（30%）但**特異性 > 95%**
- 陽性幾乎就是 SLE

### 活動度監測

**補體 C3、C4**
- **下降代表活躍中**（消耗）
- C3 < 90、C4 < 10 警訊
- 治療有效 → 上升

**dsDNA 效價趨勢**
- 上升預示復發
- 提前 1–6 個月可看出

**ESR / CRP**
- ESR 升 = 活躍
- **CRP 在純 SLE 反而不太升**（升要懷疑感染）

### 其他特異性抗體

**anti-SSA / SSB（Ro / La）**
- 也見於乾燥症
- 孕婦陽性 → 胎兒先天心臟阻滯風險
- 光敏感、亞急性皮膚狼瘡相關

**anti-RNP**
- 高效價 + 重疊症狀 → MCTD 混合性結締組織病

**抗磷脂抗體（APL）**
- anti-cardiolipin、β2-GP1、Lupus anticoagulant
- 血栓、流產風險
- 持續陽性才有意義

**抗組蛋白抗體**
- 藥物性狼瘡

### 系統評估

**血液**
- CBC：貧血、白血球低、血小板低（自體免疫性）
- 直接 Coombs 試驗

**腎臟**
- 尿液常規、尿沉渣
- 24 小時尿蛋白
- 血清肌酸酐、eGFR

**追蹤頻率**
- 穩定期：每 3 個月
- 復發或調藥：每 1 個月
- 孕期：每月

## 可以怎麼做

- 📋 抽血表格化整理（看趨勢）
- 🩺 補體下降 + 抗體升 → 預警
- 💊 治療後追蹤反應
- 📷 拍下症狀供回診對照

## 什麼時候要找醫師

- 抽血追蹤異常
- 補體下降 + 症狀加重
- 蛋白尿出現
- 治療後沒改善
- 想懷孕（要評估抗體狀態）

> 抽血**配合臨床看才有意義**——醫師看趨勢，不是單一數字。**規律追蹤是 SLE 長期穩定的關鍵**。
