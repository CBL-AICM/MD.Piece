# 第二型糖尿病併發症：心血管最重要、腎眼神經一起防

## 一句話懂

第二型糖尿病死因**七成是心血管**——不是「血糖控制不好那麼簡單」。控糖 + 控壓 + 控脂 + 戒菸 + 適當運動才能整體防護。

## 想多了解

### 心血管（首要！）
- 心肌梗塞、中風風險升 2–4 倍
- 比一般人**早 10–15 年**發生
- **SGLT-2、GLP-1 證實降心血管死亡**
- LDL 目標 < 100（已心血管病 < 70）
- 血壓 < 130/80
- 阿斯匹靈（個別化）

### 腎臟
- 糖尿病腎病變
- 進展為慢性腎病、洗腎
- 每年驗微量白蛋白尿（ACR）
- ACEi/ARB 可保護
- SGLT-2 可保護

### 眼睛
- 視網膜病變
- 黃斑部水腫
- 白內障、青光眼
- **每年眼底攝影**
- 雷射、Anti-VEGF 注射

### 神經
- 末梢神經病變（手套襪套式麻）
- 自律神經病變（姿勢性低血壓、胃輕癱、勃起功能障礙）
- 糖尿病足風險
- 神經痛可用 Pregabalin、Duloxetine

### 足部
- 全球非外傷性截肢主因
- 神經 + 血管雙重打擊
- **每天檢查腳**
- 不赤腳、合腳鞋
- 任何傷口立刻處理

### 牙齒
- 牙周病嚴重
- 互為惡性循環
- **每 3 個月洗牙**

### 感染
- 抵抗力差
- 泌尿道、皮膚、肺
- 流感、肺炎、皰疹疫苗別省

### 心理
- 憂鬱率高 2 倍
- 「糖尿病過勞」（diabetes burnout）
- 必要時諮商

### 認知
- 失智症風險升
- 控糖可降風險

### 預防併發症最關鍵
- HbA1c < 7%
- LDL < 100
- 血壓 < 130/80
- 戒菸
- 規律運動
- 定期篩檢

### 篩檢時程

| 項目 | 頻率 |
| :-- | :-- |
| HbA1c | 每 3–6 個月 |
| 血壓、體重 | 每次回診 |
| 血脂 | 每年 |
| 微量白蛋白尿 | 每年 |
| 眼底攝影 | 每年 |
| 足部 | 每年 |
| 牙科 | 每 3–6 個月 |

> 第二型糖尿病的**併發症不是命定**——好好整體管理，多數可避免或延緩。
