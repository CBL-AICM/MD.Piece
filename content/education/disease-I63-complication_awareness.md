# 中風併發症：再發中風、失智、憂鬱、吸入性肺炎

## 一句話懂

中風後 **5 年再發率 30%**——但**規律治療可降至 10%**。其他要監測：失智、憂鬱、嗆咳、深靜脈血栓。

## 想多事了解

### 再發中風（最大威脅）
- 5 年內 30% 復發（不規律治療）
- 規律治療可降至 10%
- 預防核心：抗血小板（或抗凝血）+ Statin + 控壓
- AFib 患者必須抗凝血

### 失智（血管型失智）
- 中風後失智風險升
- 多次中風累加
- 控血壓、控糖、戒菸延緩

### 憂鬱（中風後憂鬱）
- 30% 中風後憂鬱
- 不只難過，影響復健動機
- SSRI 治療有效（注意藥物交互）

### 吞嚥困難 + 吸入性肺炎
- 急性期常見
- 嗆咳 → 食物進肺 → 肺炎
- 評估後決定能否經口進食
- 必要時鼻胃管 / 胃造口

### 深靜脈血栓 / 肺栓塞
- 不動 → 血栓
- 急性期預防：抗凝血、彈性襪、早期下床

### 跌倒與骨折
- 平衡差、肌力差
- 防跌策略
- 維生素 D、鈣

### 痙攣
- 中風後肌肉緊繃
- 影響復健
- 肉毒桿菌注射、抗痙攣藥

### 排泄問題
- 失禁或便秘
- 訓練、藥物

### 性功能障礙
- 多源於藥物 + 心理
- 諮詢治療

### 認知功能下降
- 注意力、記憶、執行
- 認知訓練

### 壓瘡
- 長期臥床
- 翻身、適合床墊

### 預防併發症的核心
- 規律抗血小板 / 抗凝血
- 控三高
- 戒菸
- 規律復健
- 規律追蹤

### 篩檢與追蹤

| 項目 | 頻率 |
| :-- | :-- |
| 神經科 | 3–6 個月 |
| 抽血（三高、肝腎） | 每 3–6 個月 |
| 心電圖 | 每年（找 AFib） |
| 認知評估 | 每年 |
| 憂鬱篩檢 | 每年 |
| 吞嚥評估 | 必要時 |

## 什麼時候要找醫師
- 新發神經症狀
- 認知變差
- 持續低落、自殺念頭
- 反覆嗆咳、發燒
- 反覆跌倒
- 生活品質明顯下降

> 中風後**規律追蹤多面向** —— 預防再發、處理併發症 = 延長好生活。
