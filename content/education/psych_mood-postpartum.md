# 產後憂鬱：不是不愛孩子

## 一句話懂

很多媽媽生完孩子兩週內會掉淚、敏感——那是**產後情緒低落（baby blues）**，會自己過去。但**超過兩週**還持續低落、無望、想傷害自己或孩子，那是**產後憂鬱症**，要趕快就醫。

## 想多了解

### Baby blues vs 產後憂鬱
| | Baby blues | 產後憂鬱 |
| :-- | :-- | :-- |
| 何時 | 產後 3–10 天 | 任何時候，常 4–6 週 |
| 持續 | < 2 週 | > 2 週 |
| 嚴重度 | 哭、煩躁但能照顧寶寶 | 嚴重影響照顧能力 |
| 比率 | 70% 媽媽 | 10–15% 媽媽 |

### 為什麼會得
- **荷爾蒙劇烈變化**（雌激素、黃體素急速下降）
- **睡眠剝奪**（餵奶、夜醒）
- **角色衝擊**、社會孤立
- **過去有憂鬱史** → 風險高 3 倍

### 治療
- **哺乳期可用的抗憂鬱藥**：Sertraline（樂復得）為首選
- **心理治療**：CBT、IPT 對產後特別有效
- **支持團體**：產後媽媽小組

## 可以怎麼做

- 🛏 把睡眠當第一優先，請家人輪流顧夜
- 🤝 找一個能聽你抱怨的人（不批評不建議的那種）
- 🚶 每天出門散步 15 分鐘
- 📞 安心專線 **1925** / 113 婦幼專線

## 什麼時候要找醫師

- 兩週以上心情低落、易怒、無法享受寶寶
- 出現「想傷害自己或寶寶」的念頭 → **立刻就醫**
- 嚴重失眠、食慾全無
- 有「逃跑」「消失」的念頭

> **承認自己需要幫助，不是失敗，是負責任的媽媽。**台灣很多醫院有「孕產婦心理諮詢」，掛婦產或身心科都可以。
