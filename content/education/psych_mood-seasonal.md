# 季節性情緒障礙：冬天就憂鬱不是錯覺

## 一句話懂

每年**秋冬就低落、嗜睡、特別想吃甜食**，到春天又恢復——這是季節性情緒障礙（SAD），和日照變短有關，**光照治療效果很好**。

## 想多了解

### 為什麼冬天會這樣
日照短 → 視網膜接收的光線少 → 大腦的褪黑激素節律亂掉 → 血清素下降 → 情緒、活動力、睡眠都受影響。

### 和典型憂鬱的差別
| | 典型憂鬱 | 冬季 SAD |
| :-- | :-- | :-- |
| 食慾 | 變差 | 嗜碳水化合物 |
| 體重 | 下降 | 上升 |
| 睡眠 | 失眠 | 嗜睡 |
| 模式 | 任何時候 | 秋冬發作 |

### 三個有效治療
1. **光照治療**：每早 30 分鐘 10,000 lux 燈箱（網購可得）
2. **抗憂鬱藥**：SSRI 在冬季前開始預防
3. **走出戶外**：早上散步 15 分鐘比窩在家更有效

## 可以怎麼做

- ☀️ 早上 30 分鐘曬太陽（陰天也算數）
- 🛏 規律作息，不要冬眠模式
- 🏃 維持運動量，別讓自己宅化
- 🍞 嗜碳水可以接受，但選擇全穀類別吃精緻甜食

## 什麼時候要找醫師

- 連續兩年以上每年秋冬憂鬱
- 影響工作、社交、家庭生活
- 想嘗試光照治療但不確定怎麼選

> 台灣冬天日照雖比北歐好，但濕冷+陰天連續多日也會誘發 SAD。**這是真實存在的疾病。**
