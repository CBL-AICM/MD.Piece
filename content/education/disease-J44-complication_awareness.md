# COPD 併發症：右心衰、肺炎、肺癌、骨鬆

## 一句話懂

COPD **不只是肺的事**——影響心、骨、肌肉、心理。**整合管理**才能延長好生活。

## 想多了解

### 肺源性心臟病
- 慢性低氧 → 肺動脈高壓 → 右心衰
- 表現：下肢水腫、頸靜脈怒張
- 居家氧氣可改善
- 利尿劑、肺擴張藥

### 反覆肺炎
- COPD 患者肺炎風險高
- 抗菌防禦差
- 預防：疫苗、戒菸、勤洗手
- 嚴重肺炎可能需住院、抗生素

### 肺癌
- COPD + 抽菸 = 肺癌高風險
- 建議**低劑量胸部 CT 篩檢**（特別重度抽菸史）
- 規律檢查
- 戒菸是最大保護

### 骨質疏鬆
- 長期類固醇、不活動、營養差
- 骨折風險高
- 維生素 D + 鈣
- 必要時 Bisphosphonates
- 規律 DXA 檢查

### 肌肉萎縮（惡病質）
- 全身無力
- 與營養、活動有關
- 肺復健 + 蛋白質補充

### 心血管疾病
- COPD 患者心血管事件高
- 共同危險因子（抽菸）
- 整合管理三高

### 憂鬱、焦慮
- 25–40% 有憂鬱
- 影響治療動機
- 抗憂鬱藥（注意藥物交互）
- 諮商

### 睡眠障礙
- 夜間缺氧、呼吸暫停
- 睡眠檢查
- 居家氧氣或 CPAP

### 認知下降
- 慢性缺氧 + 共病
- 認知活動

### 預防併發症的核心
- 戒菸
- 規律吸入劑
- 肺復健
- 疫苗
- 控共病
- 規律追蹤

### 篩檢時程

| 項目 | 頻率 |
| :-- | :-- |
| 肺功能 | 每 6–12 個月 |
| 胸腔科回診 | 每 3–6 個月 |
| 抽血 | 每年 |
| 血氧 | 每次回診 |
| 低劑量 CT（高風險） | 每年 |
| 骨密度 | 2 年 |
| 心臟超音波 | 必要時 |

### 末期 COPD
- 安寧緩和介入
- 預立醫療決定
- 家屬支持
- 嗎啡可緩解末期喘

## 什麼時候要找醫師
- 體重快速下降
- 下肢水腫
- 反覆肺炎
- 認知下降
- 持續憂鬱

> COPD **是全身性疾病**——多重器官管理才能多活幾年好生活。
