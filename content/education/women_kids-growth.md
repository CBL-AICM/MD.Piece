# 生長發育：身高體重曲線、語言發展

## 一句話懂

孩子的成長**看趨勢不看單次**——生長曲線、語言動作里程碑都有時程。**遲到 ≥ 3 個月**就要評估早療。

## 想多了解

### 生長曲線

**身高、體重、頭圍**
- 出生 → 兒童健康手冊每次量
- 看百分位（3、15、50、85、97）
- 自己軌跡穩定就 OK

**警訊**
- 掉超過 2 條百分位
- < 3 或 > 97 持續

**性早熟**
- 女 < 8 歲、男 < 9 歲開始發育
- 兒童內分泌科

### 語言里程碑

| 年齡 | 應該會 |
| :-- | :-- |
| 12 個月 | 叫爸媽（有意義）|
| 18 個月 | 10–20 字 |
| 2 歲 | 雙字組合 |
| 3 歲 | 完整句子 |
| 4 歲 | 講故事 |

**早療警訊**：遲到 ≥ 3 個月

### 動作里程碑

| 年齡 | 應該會 |
| :-- | :-- |
| 3 個月 | 趴姿抬頭 |
| 6 個月 | 翻身、扶坐 |
| 9 個月 | 爬 |
| 12 個月 | 扶站 |
| 18 個月 | 獨自走 |
| 2 歲 | 跑、上下樓梯 |

### 細動作

| 年齡 | 應該會 |
| :-- | :-- |
| 9 個月 | 拇食指捏 |
| 12 個月 | 用湯匙 |
| 2 歲 | 翻書、塗鴉 |
| 3 歲 | 畫圓 |
| 4 歲 | 剪刀 |

### 早療資源

**早期療育**
- 0–6 歲黃金期
- 健保 + 政府補助
- 兒童發展聯合評估中心

**M-CHAT 自閉症篩檢**
- 18 個月、24 個月做
- 健兒門診免費

### 兒童健保預防保健

**6 次免費**
- 出生～7 歲
- 健兒門診

### 何時早療
- 任何里程碑遲到 ≥ 3 個月
- 任何能力倒退
- 不看人、不指物
- 反覆動作

### 飲食、營養
- 母乳 6 個月
- 副食品銜接
- 全食物多樣化
- 詳見「副食品與飲食」

### 運動
- 嬰兒：趴趴時間
- 兒童：每天 ≥ 60 分鐘
- 不只在 3C 前面

## 什麼時候要找醫師
- 任何發展遲緩
- 18 個月 M-CHAT 異常
- 突然能力倒退
- 父母直覺「不對勁」

> 兒童發展**早辨識早介入**——0–6 歲黃金期最寶貴。
