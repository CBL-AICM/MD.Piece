# 糖化血色素 HbA1c：與三個月平均血糖

## 一句話懂

HbA1c 反映**過去 3 個月血糖平均**——不像單次血糖會被當下飲食影響。**篩檢、診斷、追蹤**都用得到。

## 想多了解

### 為什麼能反映 3 個月
- 紅血球壽命 120 天
- 葡萄糖會與血紅素結合
- 高血糖 → 結合多 → HbA1c 高
- 反映平均血糖（不是當下）

### 數值對照

| HbA1c | 對應平均血糖 | 解讀 |
| :--: | :--: | :-- |
| 5% | 97 | 正常 |
| 6% | 126 | 邊緣 |
| 6.5% | 140 | 糖尿病門檻 |
| 7% | 154 | 糖尿病目標 |
| 8% | 183 | 控不好 |
| 9% | 212 | 嚴重 |
| 10% | 240 | 危險 |

### 診斷糖尿病
任一即可：
- HbA1c ≥ 6.5%
- 空腹血糖 ≥ 126
- OGTT 2 小時 ≥ 200
- 隨機血糖 ≥ 200 + 症狀

### 治療目標

**一般糖尿病**
- HbA1c < 7%

**個別化**
- 老年、共病多：< 7.5–8%
- 年輕、預期壽命長、無併發症：< 6.5%
- 第一型糖尿病：< 7%（兒童 < 7.5%）
- 孕期：< 6%

### 影響準確性的情況
- 貧血、地中海貧血
- 慢性腎病
- 鐮形血球
- 紅血球輸血、流失
- 妊娠後期

→ 改用其他指標（fructosamine、CGM）

### 何時驗
- 篩檢：≥ 45 歲每 3 年；高風險每年
- 糖尿病：每 3–6 個月

### 篩檢工具
- 健保 40 歲以上每 3 年免費（有條件）

### 自我追蹤
- 規律驗 HbA1c
- 配合居家血糖機
- 進階：CGM 連續血糖

## 什麼時候要找醫師
- HbA1c 上升
- 達不到目標
- 想評估治療
- 想做 CGM

> HbA1c **像血糖的成績單**——3 個月一次，告訴你長期表現。
