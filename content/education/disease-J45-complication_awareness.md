# 氣喘併發症：氣道重塑、急性發作後遺症、職業氣喘

## 一句話懂

氣喘長期未控**會氣道重塑**——變成部分不可逆。**規律保養藥**可大幅預防。

## 想多了解

### 氣道重塑（Airway Remodeling）
- 慢性發炎導致氣管壁變厚
- 平滑肌增生
- 黏液分泌增加
- 部分**不可逆**
- 預防：規律 ICS

### 急性發作後遺症
- 嚴重發作可永久損失肺功能
- 反覆發作累積
- 預防為核心

### 致死性氣喘
- 仍有少數患者死亡
- 多源於：
  - 不當用單獨 SABA
  - 漏服保養藥
  - 延遲就醫
- 嚴重發作要急診

### 共病

**過敏性鼻炎**
- 80% 氣喘合併
- 同時治療效果好
- 鼻內類固醇 + 抗組織胺

**異位性皮膚炎**
- 「異位性三聯症」
- 同時皮膚過敏

**胃食道逆流**
- 加重氣喘
- 治療逆流可改善氣喘

**睡眠呼吸中止**
- 加重氣喘
- 篩檢 + CPAP

**鼻竇炎、鼻息肉**
- 可加重氣喘
- 嚴重需手術

**精神**
- 焦慮、憂鬱
- 影響治療配合度

### 職業氣喘
- 職場暴露誘發
- 異氰酸鹽（油漆、塑膠）
- 麵粉
- 動物
- 化學品
- 改換職務或環境

### 兒童氣喘變化
- 部分長大改善
- 部分持續成人
- 家族史 + 嚴重 + 過敏 = 持續風險高

### 孕期氣喘
- 1/3 改善、1/3 不變、1/3 惡化
- 規律保養藥（多數安全）
- 不可停藥（控不好對胎兒更糟）

### 急性發作高風險者
- 一年急性發作 ≥ 2 次
- 需口服類固醇
- 曾住院 / 插管
- 對 SABA 過度依賴
- 未規律用 ICS

### 預防併發症的核心
- **規律 ICS 不要停**
- 避過敏原
- 戒菸
- 規律疫苗
- 治共病
- 早期就醫
- PEF 監測

### 篩檢時程

| 項目 | 頻率 |
| :-- | :-- |
| 控制度評估 | 每 3 個月 |
| 肺功能 | 每年 |
| PEF | 每天（控制不佳期） |
| ACT 評分 | 每次回診 |
| 過敏原檢測 | 必要時 |

## 什麼時候要找醫師
- 急救藥用 ≥ 2 次/週
- 急性發作 ≥ 1 次/年
- 控制不好
- 共病出現
- 想評估生物製劑

> 氣喘**規律治療避免併發症**——氣道重塑是不可逆的代價。
