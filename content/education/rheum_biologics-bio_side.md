# 生物製劑副作用辨識：注射部位反應、過敏、肝腎追蹤

## 一句話懂

生物製劑**整體安全**——但要學會辨識**輕微（在家處理）vs 嚴重（立刻就醫）**的差別。**早期辨識救命**。

## 想多了解

### 常見輕微副作用

**注射部位反應**
- 紅、腫、熱、輕度痛
- 24–48 小時內消退
- 處理：冰敷、輕輕按壓
- 一般不需停藥

**輕微感冒症狀**
- 多在前幾劑出現
- 通常自行緩解

**疲倦、頭痛**
- 注射後當天
- 多次後減輕

### 中度副作用（要告知醫師）

**反覆感染**
- 上呼吸道、泌尿道
- 抗生素治療
- 評估是否需暫停或換藥

**血壓變化**
- 部分藥物有影響
- 監測

**膽固醇上升**
- IL-6 抑制劑特別
- 抽血追蹤、必要時藥物

**肝腎功能異常**
- 規律抽血
- 輕度可繼續、嚴重需暫停

### 嚴重副作用（立刻就醫）

**過敏反應**
- 全身蕁麻疹
- 嘴唇、舌頭、喉嚨腫
- 呼吸困難
- → **急診**

**輸注反應**（靜脈點滴特有）
- 多發生在前幾次
- 寒顫、發燒、紅疹、低血壓
- 多可預防性類固醇 + 抗組織胺

**嚴重感染**
- 高燒 ≥ 38.5°C
- 持續咳嗽、咳血
- 嚴重腹瀉、血便
- 皮膚化膿擴散
- → **立刻就醫**

**結核再活化**
- 持續咳嗽 ≥ 2 週
- 夜間盜汗
- 體重減輕
- → **胸腔科 + 風濕科**

**B 肝再活化**
- 黃疸、茶色尿、噁心
- 抽血肝指數飆升
- → **立刻肝膽腸胃科**

**血液問題**
- 不明瘀青、出血
- 持續發燒
- 嚴重疲倦、蒼白
- → **抽血 CBC**

**心臟問題**
- 喘、下肢水腫
- 胸悶、心悸
- 特別 TNFi 在 NYHA III–IV 心衰

**神經系統（罕見）**
- 視力改變
- 肢體麻、無力
- 平衡感差
- 可能脫髓鞘事件 → MS-like

**腸穿孔（IL-6 特有）**
- 突發劇烈腹痛
- 持續性
- → **急診**

**惡性腫瘤**
- 皮膚變化（特別黑色素瘤）
- 不明腫塊
- 規律皮膚科檢查
- 規律癌症篩檢

### JAK 抑制劑特有警訊

**靜脈血栓**
- 單側下肢腫、痛
- 突發胸痛、喘
- → **立刻急診**

**心血管事件**
- 50 歲以上、心血管風險高者**避用 JAKi**
- 胸痛、心悸要警覺

**惡性腫瘤**
- 比 TNFi 略高
- 規律篩檢

**單純皰疹、帶狀皰疹**
- 警覺早期症狀
- 早治療

### 規律追蹤項目

**用藥前 baseline**
- CBC、肝腎、ESR/CRP、ANA、Lipid

**前 3 個月**
- 每月 CBC、肝腎

**穩定後**
- 每 3 個月 CBC、肝腎、Lipid

**每年**
- 結核篩檢（高風險者）
- 胸部 X 光
- 皮膚科

### 出現副作用怎麼辦

**輕度** → 記錄、回診告知
**中度** → 提前回診評估
**嚴重** → 急診 + 暫停用藥

### 副作用追蹤的資源
- 廠商藥物監測計畫
- 醫師回診評估
- 藥師諮詢

## 可以怎麼做

- 📓 副作用日記（時間、症狀、處理）
- 🩸 規律抽血不能漏
- 🌡 體溫計、血壓計居家備
- 🚨 急診時帶用藥單
- 📞 廠商諮詢專線存好
- 🚭 戒菸戒酒（減少風險）

## 什麼時候要急診

- 過敏性休克症狀
- 嚴重持續腹痛
- 突發胸痛、喘、單側下肢腫
- 高燒不退 + 持續惡化
- 黃疸、嚴重出血
- 中風樣症狀

> 生物製劑**整體安全可控**——學會分辨輕重，**早期辨識才能早期處理**。
