# 痛風與代謝症候群：高血壓、糖尿病、脂肪肝的連動

## 一句話懂

痛風**很少單獨來**——多數患者合併**高血壓、糖尿病、高血脂、脂肪肝**。這些一起構成「**代謝症候群**」，**心血管死亡率上升 1.5–2 倍**。

## 想多了解

### 代謝症候群定義（任 3 項）
- **腰圍**：男 ≥ 90 公分、女 ≥ 80 公分
- **血壓**：≥ 130/85
- **空腹血糖**：≥ 100 mg/dL
- **三酸甘油酯**：≥ 150 mg/dL
- **HDL**：男 < 40、女 < 50

### 共病比例
- 高血壓：60–70%
- 糖尿病：20–30%
- 高血脂：50–60%
- 脂肪肝：50%
- 慢性腎病：30%

### 連動機制
- 胰島素阻抗 → 腎臟尿酸排出減少
- 高糖、含糖飲料同時觸發痛風 + 糖尿
- 內臟脂肪 → 慢性發炎 → 多重影響
- 三酸甘油酯升高與尿酸正相關

### 整合管理重點

**高血壓**
- **避免 Thiazide 類利尿劑**（會升尿酸）
- **Losartan（ARB）有降尿酸效果**，是好選擇
- 鈣離子阻斷劑也安全
- 目標 < 130/80

**糖尿病**
- **SGLT-2 inhibitor**（Dapagliflozin、Empagliflozin）：降尿酸 + 降糖 + 減重
- 是痛風 + 糖尿病的好選擇
- HbA1c < 7%

**高血脂**
- **Fenofibrate** 既降脂又降尿酸
- Statin 也安全
- LDL 個別化目標

**脂肪肝**
- 減重是核心
- 控糖、控脂
- 戒酒

**心血管預防**
- 戒菸
- 規律運動
- 阿斯匹靈：注意—**低劑量會升尿酸**，但心血管獲益大於害（個別評估）

### 飲食整合
- **地中海飲食**：對痛風 + 代謝症 + 心血管都好
- 限制果糖、含糖飲料（一石多鳥）
- 限制紅肉、加工肉
- 多蔬菜、橄欖油、堅果

### 運動整合
- 一週 150 分鐘有氧 + 2 次阻力訓練
- 改善胰島素敏感、減重、降血壓
- 但**急性痛風期休息**（不要硬撐）

### 減重
- **緩慢減重**（每月 1–2 公斤）
- 急速減重 → 觸發痛風
- 5–10% 體重下降就有大影響

## 可以怎麼做

- 📊 全面檢查（血壓、血糖、血脂、肝、腎）
- 🩺 找一位主治整合管理（家醫科 + 風濕共照）
- 🥗 地中海飲食一招通吃
- 🏃 規律運動
- 🚭 戒菸戒含糖飲料
- 💊 與醫師討論藥物選擇（避開升尿酸的）

## 什麼時候要找醫師

- 痛風 + 任一代謝症
- 血壓藥用 Thiazide → 換藥
- 想評估 SGLT-2、Fenofibrate
- 心血管事件後（中風、心肌梗塞）

> 痛風**只是冰山一角**——下面的代謝症候群才更危險。**整合管理 = 多活 5–10 年**。
