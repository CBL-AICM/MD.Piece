# 血脂三項：總膽固醇、LDL、HDL、三酸甘油脂

## 一句話懂

血脂**不是越低越好**——但**LDL 是壞膽固醇**，要依心血管風險定目標。HDL 越高越好，TG 看代謝。

## 想多了解

### 四項指標

**總膽固醇 TC**
- 一般 < 200 mg/dL
- 不單獨判斷，要看分項

**LDL 低密度脂蛋白（壞膽固醇）**
- **依風險定目標**：

| 風險族群 | LDL 目標 |
| :-- | :-- |
| 一般成人 | < 130 |
| 高風險（多重因子） | < 100 |
| 已心血管病 | < 70 |
| 極高風險（反覆事件） | < 55 |

**HDL 高密度脂蛋白（好膽固醇）**
- 男 > 40、女 > 50
- 越高越好（運動、地中海飲食可升）

**TG 三酸甘油脂**
- < 150 mg/dL
- 升：肥胖、糖尿、酒精、家族
- > 500：胰臟炎風險

### LDL 計算 vs 直接測
- Friedewald 公式（TG < 400）
- TG 高時要直測

### 何時抽血
- 20 歲以上每 3–5 年一次
- 心血管風險高每年
- 治療中追蹤

### Statin 用藥指徵
- 已心血管病
- LDL ≥ 190
- 糖尿病 40–75 歲 + LDL ≥ 70
- 10 年心血管風險 ≥ 7.5%

### 飲食調整
- 限飽和脂肪、反式脂肪
- 增 Omega-3、纖維
- 地中海飲食可降 LDL 5–10%

### 運動效果
- 規律有氧可升 HDL、降 TG
- 對 LDL 影響有限

## 什麼時候要找醫師
- LDL 反覆異常
- 已 Statin 仍不達標
- 想評估 PCSK9 抑制劑
- 家族高膽固醇血症（基因型）

> 血脂控制**LDL 是關鍵**——目標依個人風險定，不是看「正常範圍」。
