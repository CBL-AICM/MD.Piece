# 動作發展：翻身、爬、走、跑的關鍵期

## 一句話懂

寶寶動作發展有**順序但有彈性**——重點不是某個月一定要達標，而是**順著進步**。落後 1–2 個月正常；**超過 3 個月或 12 個月還不能扶站**就該評估。

## 想多了解

### 粗動作里程碑

| 年齡 | 應該會 |
| :-- | :-- |
| **3 個月** | 趴姿抬頭 45° |
| **4–6 個月** | 翻身、扶坐 |
| **6–9 個月** | 獨坐、開始爬 |
| **9–12 個月** | 扶站、扶走 |
| **12–15 個月** | 獨自站、走幾步 |
| **18 個月** | 穩定走路、會跑 |
| **2 歲** | 上下樓梯（一階一腳） |
| **3 歲** | 單腳站 2 秒、踢球 |
| **4 歲** | 單腳跳、騎三輪車 |

### 細動作里程碑

| 年齡 | 應該會 |
| :-- | :-- |
| **3–4 個月** | 抓握玩具 |
| **6 個月** | 雙手交換物品 |
| **9 個月** | 食指拇指捏小物 |
| **12 個月** | 用湯匙、會指 |
| **18 個月** | 疊 2–3 塊積木 |
| **2 歲** | 翻書、塗鴉 |
| **3 歲** | 畫圓、扣大鈕扣 |
| **4 歲** | 畫人形、剪刀剪線 |

### 早療警訊
- **6 個月** 趴著不抬頭
- **9 個月** 不會獨坐
- **12 個月** 不能扶站
- **18 個月** 不會走
- **任何時候** 動作能力倒退、單側使用、肌肉張力異常

### 不必擔心
- 不爬就直接走（少數寶寶會這樣）
- 早產兒按矯正年齡看
- 略低於平均值但持續進步

## 可以怎麼做

- 🤸 **趴趴時間**從新生兒就開始，每天累積 30 分鐘
- 🧸 把玩具放遠一點吸引爬行
- 👟 室內赤腳練平衡
- 🪜 8 個月後鼓勵練爬樓梯
- 🚫 別過早用學步車（影響動作發展）

## 什麼時候要找醫師

- 任何里程碑遲到 ≥ 3 個月
- 兩側動作明顯不對稱
- 突然失去原有動作能力
- 軟趴趴或太僵硬

> 早療**越早越好**。寶寶動作慢不是「個性慢」——可能是肌肉張力、神經發展問題，可治療。
