# 腎功能：Creatinine、BUN、eGFR、尿蛋白

## 一句話懂

腎功能**不能只看肌酸酐**——eGFR 才是準確指標。**尿蛋白是腎病最早警訊**，靠驗尿才看得到。

## 想多了解

### Creatinine（肌酸酐）
- 肌肉代謝產物、由腎排
- 男 < 1.3、女 < 1.1 mg/dL
- 升 = 腎功能差
- 但**肌肉量大（健身者）會偏高**
- **肌肉少（年長者、女性）會被低估**

### eGFR（估算腎絲球過濾率）
- 用肌酸酐 + 年齡 + 性別計算
- **更準的腎功能指標**
- 正常 ≥ 90 ml/min/1.73m²
- < 60 持續 3 個月 = CKD

### CKD 分期

| 分期 | eGFR |
| :-: | :-- |
| 1 | ≥ 90 + 蛋白尿 |
| 2 | 60–89 + 蛋白尿 |
| 3a | 45–59 |
| 3b | 30–44 |
| 4 | 15–29 |
| 5 | < 15 |

### BUN（血尿素氮）
- 受脫水、蛋白攝取、消化道出血影響
- 與 Creatinine 一起看
- BUN/Cr > 20 → 脫水或上消化道出血
- 不單獨判斷腎功能

### 尿蛋白（最早警訊）

**ACR（白蛋白 / 肌酸酐比）**
- 隨機尿
- 正常 < 30 mg/g
- 30–300：微量
- > 300：明顯

**24 小時尿蛋白**
- 較準但麻煩
- 用於確認、追蹤

### 尿沉渣
- **紅血球**：腎絲球出血、結石、感染
- **白血球**：感染、間質性腎炎
- **圓柱體**：腎實質疾病

### 何時抽血篩檢

**每年一次**
- 糖尿病
- 高血壓
- 心血管疾病
- ≥ 65 歲
- 反覆腎結石

**特定情境**
- 用顯影劑前
- 化療前
- 用 NSAID 長期者

### 居家可做的
- 觀察尿色、泡沫
- 量血壓
- 體重
- 水腫

## 什麼時候要找醫師
- eGFR < 60 或下降
- 尿蛋白陽性
- 蛋白尿持續
- 不明水腫
- 反覆泌尿道感染

> 腎**是沉默器官**——靠抽血、驗尿早發現。**eGFR + 尿蛋白雙管齊下**。
