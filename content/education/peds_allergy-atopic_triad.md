# 異位性三聯症：氣喘、鼻過敏、異位性皮膚炎

## 一句話懂

過敏體質的孩子常**一個接一個出現**：嬰兒期異位性皮膚炎 → 幼兒期氣喘 → 學齡期鼻過敏。這叫「過敏進行曲」。**早期介入可以打斷接力**。

## 想多了解

### 為什麼會接力
- 同一個過敏體質（Th2 免疫偏向）
- 皮膚屏障壞 → 過敏原從皮膚進入 → 致敏 → 後續氣道反應
- 早期皮膚控制好，後續氣喘、鼻過敏機率降低

### 三種疾病重點

**異位性皮膚炎（嬰兒–幼兒）**
- 勤保濕、急性期類固醇藥膏
- → 詳見「異位性皮膚炎」專篇

**氣喘（幼兒–學齡）**
- 規律保養、急救藥備好
- → 詳見「兒童氣喘」專篇

**鼻過敏（學齡–青春期）**
- 鼻內類固醇 + 抗組織胺
- → 詳見「鼻過敏」專篇

### 共同照護重點
- **減塵蟎**（防蟎寢具、低濕度）
- **完全禁菸**
- **規律疫苗**（流感、肺炎鏈球菌）
- **規律運動**（氣喘控制下）
- **均衡飲食**（避免不必要食物限制）

## 可以怎麼做

- 🛏 防蟎枕套、被套
- 💧 室內濕度 < 60%
- 🧹 HEPA 吸塵器、減少絨毛玩具
- 🚭 全家禁菸
- 💉 流感疫苗每年打

## 什麼時候要找醫師

- 嬰兒期皮膚反覆嚴重
- 開始出現夜間咳嗽、運動後喘
- 早晨打噴嚏連發
- 想做過敏原檢測（IgE、過敏原貼布）

> 過敏進行曲不是命運——**早期積極控制**能讓孩子不必走完整套。
