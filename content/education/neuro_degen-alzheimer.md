# 阿茲海默失智：記憶為主、用藥與新藥 Lecanemab

## 一句話懂

阿茲海默症佔失智症 **60–70%**——大腦β類澱粉斑塊與 tau 蛋白堆積，**記憶力先壞**，逐漸影響其他認知。早期治療能延緩退化。

## 想多了解

### 早期 vs 老化

| 正常老化 | 阿茲海默早期 |
| :-- | :-- |
| 偶忘小事，後來想起來 | **新事完全沒印象** |
| 走神回過神知道在哪 | **熟悉地方迷路** |
| 字到嘴邊忘記 | **常找不到簡單字** |
| 算錯帳會發現 | **完全不會算** |
| 心情起伏正常 | **個性明顯改變** |

### 病程
- **輕度認知障礙 MCI**（MCI → AD：每年 10% 進展）
- **早期失智**：記憶 + 工作能力下降
- **中期**：日常生活需協助
- **晚期**：完全依賴

### 診斷
- **AD-8 量表 / MMSE / MoCA**
- **MRI**：海馬迴萎縮
- **PET 影像**：β 類澱粉斑塊（高貴）
- **腦脊液生物標記**

### 治療

**舊藥**
- **Donepezil（愛憶欣）**：第一線，輕中度
- **Rivastigmine、Galantamine**
- **Memantine（憶必佳）**：中重度加上去

**新藥（劃時代）**
- **Lecanemab（Leqembi）**：抗β類澱粉抗體
- 早期用可延緩 27% 退化
- 自費昂貴、有腦水腫風險
- **必須早期使用**

**非藥物**
- **認知訓練**
- **規律運動**（每週 150 分鐘）
- **地中海/MIND 飲食**
- **充足睡眠**（深睡時清除類澱粉）
- **社交活動**

### 共病處理
- 中風後失智常合併
- 憂鬱症（30%）
- 睡眠障礙

## 可以怎麼做

- 🧠 多動腦（讀書、學新技能、桌遊）
- 🚶 規律運動是最強保護因子
- 🍞 地中海飲食（魚、堅果、橄欖油、蔬果）
- 😴 7–8 小時睡眠
- 👥 維持社交活動
- 🩺 控制三高（也是失智因子）

## 什麼時候要找醫師

- 反覆問同樣問題
- 熟悉路線迷路
- 個性顯著改變
- 工作能力下降
- 算錢、判斷出錯
- AD-8 ≥ 2 分

> 阿茲海默**不是必然的老化**——早期診斷可以延長 3–5 年好品質生活。**找有失智專科的神經/精神科**最好。
