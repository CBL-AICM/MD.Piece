# 血液常規 CBC：白血球、紅血球、血紅素、血小板

## 一句話懂

CBC 是最基本抽血，看**三大系列細胞**：紅血球（運氧）、白血球（免疫）、血小板（止血）。

## 想多了解

### 紅血球系列

**Hb 血紅素**（女 12–16、男 13–17 g/dL）
- 低 = 貧血
- 高 = 真性紅血球增多、抽菸、脫水

**RBC、Hct**：相關指標

**MCV 平均紅血球體積**
- < 80：小球性（缺鐵、地中海貧血）
- 80–100：正常球
- > 100：大球性（葉酸 / B12 缺、酒精、肝病）

### 白血球系列

**WBC 白血球總數**（4–10×10³/μL）
- 升：感染、發炎、白血病、類固醇
- 降：病毒、骨髓問題、化療、自體免疫

**分類**
- **嗜中性球**：細菌感染↑
- **淋巴球**：病毒感染↑
- **單核球**：慢性感染↑
- **嗜酸性球**：過敏、寄生蟲、自體免疫↑
- **嗜鹼性球**：少見

### 血小板（150–400×10³/μL）

**低（出血風險）**
- < 50：明顯出血傾向
- < 20：自發性出血
- 原因：免疫性血小板減少、脾腫大、化療、登革熱

**高（血栓風險）**
- 反應性（感染、發炎、缺鐵）
- 骨髓增生疾病

### 警示組合
- WBC 高 + 嗜中性球高 → 細菌感染
- 三系列都低 → 骨髓問題
- 不明大球性貧血 → 查 B12、葉酸
- 反覆瘀青 + 血小板低 → 血液科

### 看診注意
- 帶歷次紀錄看趨勢
- 單次異常可能誤差
- 配合症狀解讀

## 什麼時候要找醫師
- 反覆異常
- 持續貧血
- 突然瘀青、出血
- 不明發燒 + 白血球異常

> CBC **是健康總體檢的核心**——三系列說的是免疫、運氧、止血的故事。
