# 藥物交互作用：葡萄柚、保健食品、酒類

## 一句話懂

藥物交互作用**不限於處方藥** —— 食物、保健食品、酒類都可能有大影響。**看診帶完整清單**最重要。

## 想多了解

### 食物

**葡萄柚（最知名）**
- 抑制 CYP3A4 酵素
- 多種藥物濃度上升 → 副作用
- **避開**：Statin（部分）、CCB（Amlodipine、Felodipine）、免疫抑制劑、部分抗精神病
- 整顆 / 果汁都會

**綠葉蔬菜（含維生素 K）**
- 影響 Warfarin
- **不必完全避開**，但量要穩定
- 突然多吃或少吃都會擾動

**牛奶 / 高鈣食物**
- 與四環素、Levothyroxine 結合 → 吸收差
- 隔開 2–4 小時

**香蕉、橘子（高鉀）**
- 與 ACEi/ARB、Spironolactone 合 → 高血鉀
- CKD 患者特別

### 保健食品 / 草藥

**銀杏**
- 抗凝血效果
- 與 Aspirin、抗凝血藥 → 出血風險

**人參**
- 影響 Warfarin
- 干擾血糖、血壓

**聖約翰草**
- 強誘導 CYP 酵素
- **降低**多種藥物濃度（避孕藥、抗排斥、抗病毒）
- 抗憂鬱藥併用 → 血清素症候群

**魚油**
- 高劑量增出血
- 與抗凝血藥小心

**葉黃素、維生素 E、C**
- 高劑量影響其他藥物

### 酒精

**+ NSAID**：胃出血風險升
**+ Acetaminophen**：肝毒性升
**+ 安眠藥 / 鎮靜劑**：呼吸抑制
**+ 抗憂鬱**：嗜睡、肝負擔
**+ 降血糖藥**：低血糖
**+ 降壓藥**：頭暈

### 處方藥之間

**常見組合**
- Warfarin + 抗生素：出血風險
- ACEi + Spironolactone：高血鉀
- Statin + CYP 抑制劑：肌肉病變
- 多種抗血小板：出血

### 看診注意

**帶完整清單**
- 處方藥（含眼藥水、外用）
- 自費藥
- 中藥、保健食品
- 偶爾才吃的藥

**多家醫院**
- 注意重複用藥
- 使用健保卡、雲端藥歷

**藥師諮詢**
- 出院前
- 領藥時
- 不確定時主動問

## 什麼時候要找醫師 / 藥師
- 開新藥前
- 想加保健食品
- 出現不明副作用
- 多家醫院看診
- 使用酒精問題

> 藥物交互**不只處方藥**——食物、補品、酒都可能有影響。**整體看才安全**。
