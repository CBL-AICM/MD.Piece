# 血管型失智：與中風相關，控好血管因子

## 一句話懂

血管型失智佔失智症 **15–20%**，僅次於阿茲海默。是**反覆小中風或慢性腦缺血**累積造成的。**和阿茲海默不同**——預防可控，**控好血管因子就能擋下來**。

## 想多了解

### 與阿茲海默的差別

| | 阿茲海默 | 血管型 |
| :-- | :-- | :-- |
| 病程 | 漸進、平滑 | **階梯式惡化**（每次中風後跳一階） |
| 早期 | 記憶為主 | 執行功能、注意力為主 |
| 影像 | 海馬迴萎縮 | 多發性梗塞、白質病變 |
| 治療 | 抗失智藥為主 | **控血管因子為主** |

### 兩種類型
- **多發性梗塞型**：反覆小中風累積
- **皮質下缺血型**：慢性小血管病變
- 常**混合型**（同時有阿茲海默）

### 危險因子
- **高血壓**（最重要）
- 糖尿病
- 高膽固醇
- 心房顫動
- 抽菸
- 已有中風或 TIA

### 治療
- **積極控血管因子**（核心！）
- **抗血小板**（如有中風史）
- **抗失智藥**（Donepezil、Memantine）效果不如阿茲海默
- **復健**（語言、認知、肢體）
- **憂鬱症處理**（常合併）

### 預後
- 比阿茲海默**進展可能更快**（每次中風跳一階）
- **但控制三高可大幅延緩**
- 早期介入差異很大

## 可以怎麼做

- 📊 血壓 < 130/80（嚴格控制）
- 💊 抗血小板規律服藥
- 🚭 完全戒菸
- 🥗 地中海/DASH 飲食
- 🚶 每週 150 分鐘運動
- 🧠 認知活動 + 社交

## 什麼時候要找醫師

- 中風後出現認知功能下降
- 階梯式惡化的失智症狀
- 反覆 TIA / 暈眩
- 多發性中風家族史

> 血管型失智**最大優勢是「可控」** ——不像阿茲海默無法避免，**控好三高真的能擋下來**。
