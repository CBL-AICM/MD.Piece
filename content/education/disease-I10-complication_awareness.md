# 高血壓併發症：心、腦、腎、眼的長期影響

## 一句話懂

高血壓**慢慢傷害器官**——多數人發現時已有變化。但**每降 10 mmHg**：中風 ↓ 30%、冠心病 ↓ 20%、心衰 ↓ 40%。**控好血壓救器官**。

## 想多了解

### 心臟
- **左心室肥厚**：心臟代償變厚
- **冠心病、心肌梗塞**
- **心衰竭**
- **心房顫動**
- 影像：心電圖、心臟超音波

### 腦
- **缺血性中風**（最常見）
- **出血性中風**
- **血管型失智**
- **多發性腔隙梗塞**
- 高血壓控好可避免大半

### 腎臟
- **慢性腎病**
- 進展為洗腎
- 蛋白尿、肌酸酐升
- 早期常無症狀
- 篩檢：尿蛋白、eGFR

### 眼睛
- **高血壓視網膜病變**
- 視力模糊、視野缺
- 眼底攝影可看
- 嚴重出血、視神經傷害

### 大血管
- **動脈硬化**全身性
- **主動脈剝離**（致命急症）
- **周邊動脈疾病**（腿部循環）
- **腹主動脈瘤**

### 預防併發症的 KEY
- **血壓 < 130/80**（一般）
- 心衰、糖尿、慢性腎病更嚴
- 控膽固醇、控糖
- 戒菸
- 規律運動
- 規律服藥不要停

### 篩檢時程

| 檢查 | 頻率 |
| :-- | :-- |
| 血壓 | 居家每天、看診每次 |
| 抽血 | 每年（肌酸酐、血脂、血糖） |
| 尿蛋白 | 每年 |
| 心電圖 | 每年 |
| 心臟超音波 | 必要時 |
| 眼底攝影 | 每 1–2 年 |

### 高血壓 + 共病
- 糖尿病：血壓 < 130/80，併發症加乘
- 心衰：血壓控制更嚴
- 慢性腎病：ACEi/ARB 必用
- 中風後：血壓控制次發預防
- 心房顫動：抗凝血藥同時用

### 老年高血壓
- 多單純收縮高
- 目標可放寬（< 140 較安全，但 SPRINT 顯示更嚴格仍有益）
- 避免立姿低血壓造成跌倒

## 什麼時候要找醫師
- 血壓不達標
- 出現器官受損跡象
- 共病需整合管理
- 老年血壓波動大

> 高血壓**安靜但有毒**——20 年沒控好就會交出器官。**今天的努力是 70 歲的健康**。
