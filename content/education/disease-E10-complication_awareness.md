# 第一型糖尿病併發症認知：眼、腎、神經、心血管

## 一句話懂

長期高血糖會傷小血管 + 大血管——影響眼、腎、神經、心臟。**DCCT 研究證實：嚴格控糖可降併發症 50%**。

## 想多了解

### 眼睛：糖尿病視網膜病變
- 全球失明主因之一
- 多無症狀直到嚴重
- **每年眼底攝影**
- 可雷射、Anti-VEGF 注射
- 嚴重需玻璃體切除術

### 腎臟：糖尿病腎病變
- 進展為慢性腎病、洗腎
- **每年驗微量白蛋白尿**（ACR）
- 控好血壓 + ACEi/ARB 是關鍵
- 蛋白尿出現 = 警訊

### 神經：糖尿病神經病變
- 末梢神經病變（手套襪套式麻）
- 自律神經（姿勢性低血壓、胃輕癱）
- 足部護理重要（防糖尿病足）
- 神經痛可用 Pregabalin、Duloxetine

### 心血管
- **動脈硬化**加速
- 心肌梗塞、中風風險升 2–4 倍
- 比一般人**早 10–15 年**發生
- 控糖 + 控血壓 + 控膽固醇 + 戒菸

### 足部問題
- 神經病變 → 感覺差不知受傷
- 血液循環差 → 不易癒合
- **每天檢查腳**
- 不赤腳、合腳鞋
- 任何傷口立刻處理
- 截肢主因可預防

### 牙齒
- 糖尿病者牙周病嚴重
- 互為惡性循環
- **每 3 個月洗牙、定期看牙**

### 心理
- 憂鬱率高（2–3 倍）
- 飲食失調風險（特別年輕女性）
- 必要時心理介入

### 預防併發症最關鍵
- **HbA1c < 7%**（個別化）
- **血壓 < 130/80**
- **LDL < 100 mg/dL**
- 戒菸
- 規律運動
- 規律篩檢

### 篩檢時程

| 檢查 | 頻率 |
| :-- | :-- |
| HbA1c | 每 3 個月 |
| 血壓 | 每次回診 |
| 血脂 | 每年 |
| 微量白蛋白尿 | 每年 |
| 眼底攝影 | 每年 |
| 足部檢查 | 每年 + 每次看診目視 |
| 牙科 | 每 6 個月 |

> 第一型糖尿病**併發症不是命定**——好好控糖，多數可避免或延緩。
