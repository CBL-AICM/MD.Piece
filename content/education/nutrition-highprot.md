# 高蛋白飲食：術後、銀髮、肌少症

## 一句話懂

老化、術後、肌少症患者需**比一般人多吃蛋白**——**1.0–1.2 g/kg/day**，**每餐 25–30 g** 較有效，**配合阻力訓練**才能變肌肉。

## 想多了解

### 各族群建議量

| 族群 | g/kg/day |
| :-- | :--: |
| 一般成人 | 0.8 |
| 老年人 | 1.0–1.2 |
| 阻力訓練 | 1.4–1.6 |
| 術後恢復 | 1.2–1.5 |
| 肌少症治療 | 1.2–1.5 |
| 透析患者 | 1.2–1.5 |

例：60 公斤老人 → 60–72 g/day

### 高蛋白食物（25 g 蛋白等量）

- 雞胸肉 100 g
- 鮭魚 130 g
- 雞蛋 4 顆
- 嫩豆腐 250 g
- 牛奶 700 ml
- 優格 350 g

### 每餐分配

**重要：均勻分配**
- 每餐 25–30 g
- 比集中一餐效率高
- 早餐特別容易缺

**早餐高蛋白範例**
- 蛋 2 顆 + 牛奶 + 全麥土司
- 優格 + 堅果 + 莓果

### 配合阻力訓練

**蛋白質 + 訓練 = 肌肉**
- 單吃蛋白：脂肪
- 單訓練：肌肉量小幅
- 兩者配合：明顯增肌

**訓練後 30–60 分鐘**
- 補蛋白窗口期
- 喝高蛋白牛奶、雞胸便當

### 老人特別注意

**「同化阻抗」**
- 老人對蛋白同化效率差
- 需更多量（1.0–1.2 g/kg）
- 每餐 25–30 g 較有效

**牙口差怎辦**
- 蛋、奶、豆腐、絞肉
- 蛋白粉
- 軟質料理

### CKD 的特例

**第 1–3 期**
- 0.6–0.8 g/kg（適量）
- 不必高蛋白

**透析期**
- 反而需 1.2–1.5
- 失去蛋白於透析液

### 蛋白粉

**何時用**
- 飲食不夠
- 訓練後補充
- 食慾不好的老人

**選擇**
- 乳清（吸收快）
- 酪蛋白（持久）
- 大豆（植物性）

**注意**
- 過量沒益處
- 看標示成分

### 動物性 vs 植物性

**動物性**
- 完整氨基酸
- 蛋、奶、肉、魚

**植物性**
- 多互補（豆 + 穀）
- 大豆是完整蛋白
- 高纖、低飽和脂肪

### 過量的副作用
- 腎負擔（健康人無慮，但高蛋白 > 2 g/kg 長期需評估）
- 鈣流失（極端高蛋白）
- 脫水（蛋白代謝產生尿素）

## 什麼時候要找營養師
- 術後恢復計畫
- 肌少症介入
- CKD 個別化
- 想增肌

> 高蛋白**不是越多越好**——分配 + 配合訓練 = 真正的肌肉。
