# 緊張型頭痛：肩頸僵硬與生活壓力

## 一句話懂

最常見的頭痛——**雙側、像被緊箍咒**綁住、悶悶脹脹。**不像偏頭痛會抽動**，活動也不會更痛。**九成以上的人**都經歷過。

## 想多了解

### 和偏頭痛的差別

| | 偏頭痛 | 緊張型 |
| :-- | :-- | :-- |
| 部位 | 單側 | 雙側 |
| 特性 | 搏動 | 緊箍、悶 |
| 噁心 | 常 | 少 |
| 怕光怕聲 | 是 | 不明顯 |
| 活動 | 加重 | 不變 |

### 為什麼會痛
- **顱周肌肉緊繃**（頭皮、頸後）
- **中樞痛覺敏感化**（慢性化後）
- 與壓力、姿勢、咬合有關

### 治療
- **急性**：NSAID、Acetaminophen
- **慢性**（每月 ≥ 15 天）：
  - **Amitriptyline 低劑量**：第一線預防
  - **熱敷、按摩、針灸**有幫助
  - **CBT 認知行為治療**
  - **生物回饋訓練**

### 藥物過量性頭痛
- 一週吃止痛藥 ≥ 3 次連續 3 個月
- 反而越吃越頭痛
- **必須戒藥**才能改善

## 可以怎麼做

- 💆 每天伸展肩頸 5 分鐘
- 🪑 工作姿勢調整（螢幕高度、椅子）
- 😴 規律睡眠
- 🧘 深呼吸、正念冥想
- 🏃 規律有氧運動

## 什麼時候要找醫師

- 一週發作 ≥ 2 次
- 止痛藥越吃越多
- 影響工作、睡眠
- 出現新型態頭痛
- 突發劇烈頭痛 → **急診**

> 緊張型頭痛**多是良性**——但慢性化後會嚴重影響生活。**不要靠止痛藥硬撐**，找出根本原因。
