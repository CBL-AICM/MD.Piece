# 甲狀腺功能：TSH、T3、T4 解讀

## 一句話懂

**TSH 是甲狀腺功能最敏感指標**——TSH 高 = 甲狀腺功能低、TSH 低 = 甲狀腺功能亢進。配合 T4、T3 確認。

## 想多了解

### TSH 甲狀腺刺激素
- 由腦下垂體分泌
- 「**反向**」反映甲狀腺
- 正常 0.4–4.5 mIU/L

### T4（甲狀腺素）
- 主要循環荷爾蒙
- Free T4 較準確

### T3
- 活性形式
- 多由 T4 轉化
- 甲亢時更敏感

### 解讀

| TSH | T4 | 解讀 |
| :-- | :-- | :-- |
| 高 | 低 | 原發性甲低 |
| 高 | 正常 | 次臨床甲低 |
| 低 | 高 | 原發性甲亢 |
| 低 | 正常 | 次臨床甲亢 |
| 低 | 低 | 中樞性甲低（罕見）|

### 甲狀腺亢進

**症狀**
- 心悸、手抖
- 怕熱、流汗
- 體重減、食慾增
- 失眠、焦慮
- 月經減少
- 凸眼（Graves'）

**原因**
- Graves 病（自體免疫）
- 結節性甲亢
- 急性甲狀腺炎

**治療**
- Methimazole、PTU
- 放射碘
- 手術

### 甲狀腺低下

**症狀**
- 疲倦、嗜睡
- 怕冷
- 體重增
- 便秘
- 皮膚乾、頭髮稀
- 月經多
- 抑鬱

**原因**
- 橋本氏甲狀腺炎（自體免疫）
- 碘缺乏（少見）
- 放射碘後
- 藥物（Lithium、Amiodarone）

**治療**
- Levothyroxine（Eltroxin）
- 早晨空腹服
- 規律抽血調劑量

### 篩檢時機
- 不明疲倦、體重變化
- 心悸、手抖
- 月經不規則
- 不明憂鬱
- 孕期、產後
- 抗甲狀腺抗體陽性家族

### 抗體
- **anti-TPO**：橋本氏
- **anti-Tg**：橋本氏
- **TRAb / TSI**：Graves 病

### 甲狀腺結節
- 超音波評估
- 必要時細針抽吸（FNA）

## 什麼時候要找醫師
- 持續疲倦、心悸
- 不明體重變化
- TSH 異常
- 摸到頸部腫塊
- 孕期甲狀腺評估

> TSH **是甲狀腺功能的方向盤**——一個指標看大方向。
