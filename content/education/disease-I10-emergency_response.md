# 高血壓緊急應變：高血壓急症與危症的辨別

## 一句話懂

血壓 ≥ 180/120 **不必急著降到正常**——重點是看有沒有「**器官受損**」。**有 → 立即急診（危症）；沒有 → 門診調整（急症）**。

## 想多了解

### 高血壓危症（Hypertensive Emergency）
血壓 ≥ 180/120 + **器官受損**：
- **胸痛**（心肌梗塞、主動脈剝離）
- **嚴重頭痛 + 視力變化**（腦病變）
- **單側無力、講話不清**（中風）
- **嚴重喘 + 端坐呼吸**（肺水腫）
- **腰痛 + 血尿**（急性腎傷害）
- **抽搐、意識不清**

→ **立刻打 119 急診**

### 高血壓急症（Hypertensive Urgency）
血壓 ≥ 180/120 但**無器官受損**：
- 多無症狀或輕微頭痛
- → 24–48 小時內看門診
- 不需急診
- **不需急速降壓**（會反而傷器官）

### 居家高血壓飆高怎麼辦

**不要慌**
- 休息 5 分鐘再量
- 確認量測正確（坐姿、手放心臟高度）
- 量兩次取平均

**有以下症狀 → 急診**
- 上述危症任一項

**無症狀 → 慢慢處理**
- 確認是否漏服降壓藥 → 補
- 諮詢醫師加量
- 24 小時內回診
- **不要自己加 Captopril 舌下含服**（過時做法、會降太快）

### 預防危症
- 規律服藥不要停
- 監測居家血壓
- 控制壓力
- 戒菸戒酒
- 不擅自停藥

### 急性主動脈剝離（最致命）
- 突發**撕裂樣**胸痛 / 背痛
- 痛感「移動」
- 多在高血壓控制不佳者
- 立刻急診

### 妊娠高血壓
- 子癇前症、子癇症
- 收縮壓 ≥ 160 + 蛋白尿
- → 產科急診

## 什麼時候要急診
- 高血壓 + 任一危症症狀
- 突發劇烈胸 / 背 / 頭痛
- 神經學異常
- 視力急遽下降
- 嚴重喘

> 高血壓飆高**不必都急診**——但**有器官症狀就別猶豫**。
