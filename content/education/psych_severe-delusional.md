# 妄想症：固定的、局部的錯誤信念

## 一句話懂

患者堅信某件事是真的（被監視、被下毒、配偶外遇、自己有絕症），**證據再清楚也說不動**。但生活其他面向相對正常——這是妄想症，和思覺失調有差別。

## 想多了解

### 常見類型
- **被害型**：覺得被跟蹤、被陷害（最常見）
- **嫉妒型**：堅信配偶外遇
- **情愛型**：相信某名人愛自己（追星到極端）
- **誇大型**：自認是先知、有神聖使命
- **身體型**：堅信身上有蟲、發臭、變形

### 和思覺失調的差別
| | 思覺失調 | 妄想症 |
| :-- | :-- | :-- |
| 幻覺 | 常見 | 少 |
| 妄想範圍 | 廣、奇怪 | 局部、可能是真的 |
| 其他功能 | 全面退化 | 大致正常 |
| 起病 | 多在 20–30 歲 | 較晚（40+） |

### 治療
- **抗精神病藥**（Risperidone、Aripiprazole 低劑量）
- 治療反應比思覺失調慢、效果不如思覺失調明顯
- **建立信任的醫病關係**比快速說服更重要
- 心理治療幫助處理因妄想造成的人際衝突

## 可以怎麼做（家屬角度）

- 🚫 **不要直接否定妄想**——會破壞關係
- 🤝 認可情緒（「你一定很痛苦」）但不認可內容
- 📋 記錄具體影響生活的事件（為日後就醫備用）
- 💊 鼓勵服藥（妄想不會「自己想開」）

## 什麼時候要找醫師

- 妄想持續一個月以上
- 因妄想做出影響家庭的決定
- 出現傷害他人的計畫 → **立即就醫**

> 妄想症患者通常不認為自己有病——**家屬的耐心與堅持很關鍵**。
