# RA 抽血指標：RF、anti-CCP、ESR、CRP 怎麼看

## 一句話懂

RA 沒有「100% 確認」的抽血——要結合症狀。**anti-CCP > RF**（特異性高）。**陰性也可能是 RA**（10–20% seronegative RA）。

## 想多了解

### RF 類風濕因子
- **第一個被發現**的 RA 抗體
- 敏感度 70%、特異度 80%
- **問題**：很多其他病也會陽性
  - 乾燥症、SLE、慢性肝炎、結核、健康老人
- 高效價更有意義

### anti-CCP 抗環瓜胺酸肽抗體
- **特異性 95%**（陽性幾乎就是 RA）
- 可在症狀出現前數年陽性
- 高效價預示骨侵蝕風險高
- **首選的 RA 確認指標**

### 雙陽性（RF + anti-CCP）
- 確診信心最高
- 預後較差、骨侵蝕快
- **要積極治療**

### 雙陰性 RA（Seronegative）
- 仍佔 RA 約 20%
- 臨床表現可能較輕
- 預後通常較好

### ESR / CRP（活動度指標）
- 活動度高 → 上升
- 治療有效 → 下降
- **不一定平行變化**：CRP 較敏感
- 用於追蹤、不用於診斷

### 其他相關抽血
- CBC：看貧血、血小板
- 肝腎功能：用藥前基線
- 尿酸：排除痛風
- ANA：排除其他自體免疫
- HBV、HCV、TB：用 DMARD 前篩檢
- HLA-DR4：早期積極型 RA

## 用藥前必驗
- 肝腎功能
- B 肝、C 肝
- 結核（IGRA 或 X 光）
- HIV
- 全血球計數

## 可以怎麼做

- 📋 留存歷次抽血給醫師看趨勢
- 🩺 抽血前不用空腹（除非合併血脂）
- 💊 不要自己解讀單一數值
- 📅 治療開始後追蹤抽血頻率（醫師指示）

## 什麼時候要找醫師

- RF 或 anti-CCP 陽性
- ESR / CRP 持續上升
- 治療中追蹤異常
- 用藥前篩檢項目陽性

> 抽血是**配角不是主角**——RA 診斷靠**抽血 + 症狀 + 影像**整合。
