# 顏面神經麻痺：貝爾氏麻痺、類固醇黃金期

## 一句話懂

一覺醒來**半邊臉不能動**：眼睛閉不起、嘴歪、喝水會漏——多是**貝爾氏麻痺**（病毒性顏面神經發炎）。**72 小時內用類固醇**恢復最快。

## 想多了解

### 典型表現
- **單側**整邊臉麻痺（額頭也不能皺）
- 眼睛閉不緊（角膜乾燥風險）
- 嘴角歪、喝水漏
- 患側味覺異常
- 同側耳後疼痛

### 重要鑑別

**貝爾氏麻痺 vs 中風**

| | 貝爾氏 | 中風 |
| :-- | :-- | :-- |
| 額頭 | 不能皺 | **可以皺**（皮層代償） |
| 手腳 | 正常 | **可能無力** |
| 起病 | 數小時–幾天 | 突發 |
| 危險 | 低 | 高 |

**所以「能不能皺額頭」是關鍵**——額頭沒事不能放心，要看手腳。

### 治療
- **類固醇**（Prednisolone）：72 小時內開始效果最好
- **抗病毒藥**：嚴重者加 Acyclovir
- **眼部護理**：人工淚液、夜間眼藥膏、眼罩（防角膜潰瘍）
- **早期復健**：被動運動、針灸、電療

### 預後
- **70% 完全恢復**（3–6 個月內）
- 15% 部分恢復
- 15% 留下長期影響
- **越早治療、越輕度恢復越好**

## 可以怎麼做

- 💊 越早就醫越好（72 小時內最關鍵）
- 👁 眼睛保護：白天人工淚液、晚上眼藥膏 + 眼罩
- 💆 對著鏡子做表情訓練（皺眉、抬眉、微笑、噘嘴）
- 🚫 不要強迫快速恢復、不要過度按摩

## 什麼時候要急診

- 突發臉麻 → 立刻急診（要先排除中風）
- 同時手腳無力、講話不清
- 反覆發作或雙側麻痺
- 頭痛、發燒合併出現
- 耳朵有水泡（可能 Ramsay Hunt 症候群）

> 顏面神經麻痺**多會好**——但**先排除中風、72 小時內用藥**是兩大關鍵。不要拖。
