# 心衰併發症：心律不整、腎衰、惡病質

## 一句話懂

心衰**不是單一器官問題**——影響全身。**心律不整、腎衰、惡病質、認知下降**都是要監測的併發症。

## 想多了解

### 心律不整
- **AFib**：30% 心衰合併
- **心室頻脈、心室顫動**：致命
- 嚴重者植入 ICD（防猝死）
- 同時治療兩者

### 腎臟（心腎症候群）
- 心衰 → 腎血流減少
- 腎衰 → 心衰加重
- ACEi/ARB、SGLT-2 雙向保護
- 規律驗 eGFR、尿蛋白

### 心衰惡病質
- 嚴重肌肉、脂肪流失
- 預後差
- 需積極營養介入
- 高蛋白、適量熱量

### 認知下降
- 心衰患者失智風險升
- 多源於腦部低灌流
- 控好心衰可延緩

### 憂鬱、焦慮
- 30% 心衰患者有
- 影響預後
- 必要時抗憂鬱
- 注意藥物交互

### 貧血、缺鐵
- 50% 心衰患者
- 影響運動耐力
- 鐵劑（口服或靜脈）治療

### 睡眠呼吸中止
- 中重度心衰常見
- CPAP 可改善
- 篩檢：睡眠檢查

### 性功能障礙
- 多源於藥物 + 心血管狀況
- 與醫師討論調整

### 預防併發症的核心
- **四大支柱規律使用**
- 控制共病（糖尿、高壓、AFib）
- 限鹽限水
- 規律運動 + 心臟復健
- 戒菸戒酒
- 規律監測

### 篩檢項目

| 項目 | 頻率 |
| :-- | :-- |
| 體重、症狀 | 每天 |
| 血壓、心率 | 每次回診 |
| 抽血（電解質、腎、BNP） | 每 3 個月 |
| 心電圖 | 每年 |
| 心臟超音波 | 每年或惡化時 |
| 24 小時尿、蛋白尿 | 每年 |

### 末期心衰選項
- 心臟移植（< 65 歲、無嚴重共病）
- 機械心室輔助裝置（LVAD）
- 安寧療護

## 什麼時候要找醫師
- 症狀加重
- 新發心律不整
- 腎功能下降
- 體重大幅變化
- 認知變化
- 想討論進階治療

> 心衰是**整體性疾病**——多器官管理才能延長好生活。
