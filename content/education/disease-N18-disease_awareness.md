# 慢性腎臟病：腎功能漸進下降，可延緩可預防

## 一句話懂

慢性腎臟病（CKD）多源於**糖尿病、高血壓**——腎功能漸進下降。**早期無症狀**，靠抽血篩檢。**SGLT-2** 是新利器，**洗腎不是必然**。

## 想多了解

### 為什麼會得
- **糖尿病**（首要）
- **高血壓**
- 慢性腎絲球腎炎
- 自體免疫（SLE 狼瘡腎炎）
- 多囊性腎病
- 反覆腎結石、感染
- 長期止痛藥

### 分期（依 eGFR）

| 期 | eGFR | 描述 |
| :-: | :-: | :-- |
| 1 | ≥ 90 + 蛋白尿 | 正常但有蛋白尿 |
| 2 | 60–89 | 輕度下降 |
| 3a | 45–59 | 中度 |
| 3b | 30–44 | 中重度 |
| 4 | 15–29 | 重度（準備透析） |
| 5 | < 15 | 末期（需透析或移植） |

### 表現
- **早期幾乎無症狀**
- **泡沫尿**（蛋白尿）
- 夜尿增加
- 後期：水腫、疲倦、貧血、皮膚癢、食慾差

### 篩檢
- **每年驗**：肌酸酐、eGFR、尿蛋白（ACR）
- 高風險族群（糖尿、高血壓）特別重要

### 治療

**控基礎疾病**
- 糖尿 HbA1c < 7%
- 血壓 < 130/80（蛋白尿者更嚴）

**保腎藥**
- **ACEi / ARB**（蛋白尿者）
- **SGLT-2 抑制劑**（劃時代，糖尿與非糖尿都有效）
- **GLP-1 RA**（糖尿病合併）
- **MRA Finerenone**（糖尿病腎病變）

**飲食**
- 適量蛋白
- 低鹽
- 看狀況限磷、鉀

**避免傷腎物**
- NSAID 慎用
- 顯影劑前評估
- 部分中草藥（馬兜鈴酸）

### 透析準備
- eGFR < 30 開始評估
- 血液透析或腹膜透析
- 移植候選者評估

### 健保涵蓋
- 多數 CKD 用藥健保給付
- 透析、移植給付

### 治療成功率
- 早期介入可大幅延緩進展
- SGLT-2 改變預後
- 不是所有 CKD 都會走到洗腎

> CKD **早期可控制可延緩**——關鍵是早篩檢、早控糖控壓、早用 SGLT-2。
