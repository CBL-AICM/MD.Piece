# 孕期保健：葉酸、產檢時程、體重增加

## 一句話懂

孕期保健 = **孕前準備 + 規律產檢 + 合理體重 + 健康生活**。準備好可以健康懷孕。

## 想多了解

### 孕前準備

**葉酸**
- 受孕前 3 個月 + 孕初期
- 400 μg/day（一般）
- 4 mg/day（神經管缺陷家族史）
- 預防神經管缺陷

**抽血**
- B 肝、梅毒、HIV
- 德國麻疹抗體
- 地中海貧血
- 三高

**疫苗**
- 麻疹腮腺炎德國麻疹（活性，孕前打）
- B 肝、Tdap

**慢性病評估**
- 糖尿、高血壓、SLE 等
- 提前控好
- 藥物調整為孕期可用

### 產檢時程（健保給付 14 次）

| 週數 | 檢查 |
| :-- | :-- |
| 8–12 週 | 確認懷孕、超音波 |
| 11–14 週 | 第一孕期唐氏症篩檢 |
| 16–18 週 | 羊膜穿刺（高風險） |
| 16–22 週 | 第二孕期母血唐氏 |
| 20–24 週 | 高層次超音波（自費） |
| 24–28 週 | 妊娠糖尿病篩檢 |
| 32 週 | B 肝、HIV 重驗 |
| 35–37 週 | B 群鏈球菌篩檢 |
| 後期 | 每 1–2 週 |

### 體重增加建議

| 孕前 BMI | 建議增加 |
| :-- | :-- |
| < 18.5 | 12.5–18 kg |
| 18.5–24 | 11.5–16 kg |
| 24–27 | 7–11.5 kg |
| ≥ 27 | 5–9 kg |

**多胞胎**：增加更多

### 飲食

**多吃**
- 蛋白質
- 鈣、鐵
- Omega-3（DHA）
- 葉酸食物

**避免**
- 生食（生魚、生蛋、未殺菌乳）
- 高汞魚（鯊、旗魚）
- 酒精（任何量都不安全）
- 過量咖啡因（< 200 mg/day）

### 運動
- 規律低衝擊
- 游泳、瑜珈、走路
- 避免高摔倒風險

### 注意警訊（立刻就醫）
- 嚴重出血
- 嚴重腹痛
- 持續嘔吐 > 24 小時
- 嚴重頭痛 + 視力變化（疑子癇前症）
- 胎動明顯減少
- 破水

### 妊娠糖尿病
- 24–28 週篩檢
- 75 g OGTT
- 控好減胎兒併發症
- 飲食、運動、必要時胰島素

### 子癇前症
- 高血壓 + 蛋白尿
- 嚴重頭痛、視力變化、腹痛警訊
- 阿斯匹靈預防（高風險）

### 慢性病媽媽

**糖尿病**
- HbA1c < 6.5% 目標
- 胰島素為主
- 加強血糖監測

**高血壓**
- 換成 Methyldopa、Labetalol、Nifedipine
- ACEi/ARB 禁用

**SLE**
- 穩定 6 個月以上
- HCQ 留下
- 看抗體狀態評估

**癲癇**
- 不可自行停藥
- 部分藥需替換

## 什麼時候要急診
- 上述警訊任一
- 任何擔心的症狀

> 孕期**準備 + 規律產檢**=健康懷孕。慢性病媽媽計畫好也可以。
