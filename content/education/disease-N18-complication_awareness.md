# 慢性腎臟病併發症：心血管、貧血、骨病、營養

## 一句話懂

CKD 併發症**全身性**——**心血管事件是首要死因**（不是洗腎）。也要管貧血、骨病、營養、感染。

## 想多了解

### 心血管疾病（首要！）
- CKD 患者**多死於心血管而非腎衰**
- 心肌梗塞、中風、猝死風險升
- 預防：控血壓、Statin、SGLT-2、戒菸、運動

### 貧血
- 腎臟分泌 EPO 減少
- 嚴重時影響運動、心血管
- 治療：紅血球生成素 + 鐵劑
- 目標 Hb 10–11.5 g/dL

### 礦物質與骨病（CKD-MBD）
- 磷、鈣、維生素 D、副甲狀腺素失衡
- 骨折、血管鈣化風險
- 治療：限磷、磷結合劑、活性維生素 D
- DXA 骨密度檢查

### 酸血症
- 腎排酸能力差
- 影響骨頭、肌肉
- 碳酸氫鈉補充

### 營養不良
- 食慾差 + 飲食限制
- 蛋白質流失
- 體重下降、肌肉萎縮
- 營養師諮詢

### 感染
- 免疫力下降
- 流感、肺炎、泌尿道感染
- 疫苗：流感、肺炎鏈球菌、COVID-19、帶狀皰疹、B 肝

### 高血壓（互為因果）
- CKD 引起高血壓、高血壓加重 CKD
- 嚴格控制 < 130/80

### 神經系統
- 神經病變（手腳麻）
- 認知下降
- 末期：尿毒性腦病變

### 心理
- 憂鬱率 30%
- 「等待透析」的焦慮
- 必要時諮商、藥物

### 性功能 / 生育
- 男：勃起功能障礙
- 女：月經異常、生育力下降
- 透析後改善有限
- 移植後可恢復

### 痛癢
- 末期常見
- 影響睡眠
- 抗組織胺、UV-B、Difelikefalin（新藥）

### 預防併發症的核心

**保腎**
- 規律 ACEi/ARB、SGLT-2
- 控血壓、控糖
- 避傷腎物

**護心**
- Statin
- 戒菸
- 運動

**防貧血**
- 規律驗 Hb
- EPO + 鐵

**防骨病**
- 限磷、磷結合劑
- 維生素 D
- 規律 DXA

**疫苗**
- 全套打齊

### 篩檢時程

| 項目 | 頻率 |
| :-- | :-- |
| 抽血（腎、電解質、貧血、磷鈣） | 每 1–3 個月 |
| 尿蛋白 | 每 3 個月 |
| iPTH（後期） | 每 6 個月 |
| HbA1c | 每 3 個月 |
| 心電圖 | 每年 |
| 心臟超音波 | 每 1–2 年 |
| DXA | 每 2 年 |

### 末期選項
- 血液透析
- 腹膜透析
- 腎臟移植
- 安寧療護（非透析選擇）

## 什麼時候要找醫師
- 任一併發症出現
- 心血管症狀
- 不明貧血
- 體重快速下降
- 想評估透析或移植

> CKD 併發症**整體管理才能延長好生活**——**不只看 eGFR**。
